# PDF 변환 프롬프트 템플릿

## 개요

`.note/` 폴더의 핵심 요약 노트(`.md`)를 인쇄용 PDF로 변환하기 위한 프롬프트 정의 문서.
Web Claude AI에 MD 파일과 함께 제공하면 단일 HTML 파일을 생성하고,
브라우저에서 Ctrl+P → PDF 저장 방식으로 최종 PDF를 얻는다.

---

## 고정 설정값 (변경 불필요)

| 항목               | 값                                 |
| ------------------ | ---------------------------------- |
| 구현 방식          | HTML + CSS (브라우저 Print to PDF) |
| 용지               | A4                                 |
| 레이아웃           | 1단                                |
| 인쇄 목적          | 컬러 프린터                        |
| 디자인             | 컬러 강조                          |
| 여백 (상/하/좌/우) | 15mm / 15mm / 18mm / 40mm          |
| 우측 메모 공간     | 40mm (점선 세로선)                 |
| 본문 폰트          | Noto Sans KR (Google Fonts CDN)    |
| 코드 폰트          | D2Coding, monospace                |
| 본문 크기          | 10pt                               |
| 표 내부 크기       | 8.5pt                              |

---

## 색상 정책

| 요소             | 색상 코드 | 적용                         |
| ---------------- | --------- | ---------------------------- |
| H2 섹션 배경     | `#1a3a5c` | 진한 남색 배경 + 흰색 텍스트 |
| H3 소제목 강조선 | `#2e7bcf` | 좌측 4px 세로선              |
| 표 헤더          | `#2e7bcf` | 배경 + 흰색 텍스트           |
| 표 홀수행        | `#f0f6ff` | 연한 파랑 배경               |
| 함정선지 (❌)    | `#cc0000` | 빨간색 텍스트                |
| 코드블록 강조선  | `#2e7bcf` | 좌측 4px 세로선              |
| 코드블록 배경    | `#f5f5f5` | 연한 회색                    |
| Bold 텍스트      | `#1a3a5c` | 진한 남색                    |

---

## 페이지 규칙

- H2 뒤 첫 내용이 다음 페이지로 분리되지 않도록 처리
- 페이지 번호: 하단 중앙
- 표지: 없음
- 외부 의존성: Google Fonts CDN만 허용 (완전한 단일 HTML)

---

## 프롬프트 본문

> 아래를 복사해서 Web Claude AI에 MD 파일 내용과 함께 붙여넣는다.

---

```
아래 마크다운 파일을 인쇄용 PDF로 변환하기 위한
단일 HTML 파일(.html)을 생성해줘.
브라우저에서 열고 Ctrl+P → PDF 저장 방식으로 사용할 거야.

--- 출력 규격 ---
- 용지: A4
- 레이아웃: 1단
- 여백: 상 15mm / 하 15mm / 좌 18mm / 우 40mm (오른쪽은 메모 공간)
- 오른쪽 40mm 영역에 점선 세로선 표시

--- 폰트 ---
- 본문: Noto Sans KR (Google Fonts CDN)
- 코드블록: D2Coding, monospace
- 본문 크기: 10pt
- 표 내부: 8.5pt

--- 색상 정책 ---
- H2 섹션 제목: 진한 남색(#1a3a5c) 배경 + 흰색 텍스트
- H3 소제목: 좌측에 #2e7bcf 색상 4px 세로선
- 표 헤더 행: #2e7bcf 배경 + 흰색 텍스트
- 표 홀수행: #f0f6ff 배경, 짝수행: 흰색
- ❌ 로 시작하는 줄: 텍스트 #cc0000 (빨간색)
- 코드블록 배경: #f5f5f5, 테두리: #ddd, 좌측 4px #2e7bcf 강조선
- bold 텍스트: #1a3a5c

--- 페이지 나눔 ---

- H2 바로 뒤 내용은 같은 페이지에 유지 (break-after: avoid)

--- 인쇄 최적화 ---
- @media print 적용
- 링크 밑줄 제거
- 배경색·배경이미지 인쇄 강제 적용
  (-webkit-print-color-adjust: exact)

--- 기타 ---
- 페이지 하단 중앙에 페이지 번호 표시
- 표지 없음
- 외부 이미지 없이 완전한 단일 HTML 파일로 생성
- Google Fonts CDN만 외부 의존성으로 허용


```

---

## 사용 순서

```
1. 위 프롬프트 본문 복사
2. Web Claude AI 대화창에 붙여넣기
3. [여기에 MD 파일 내용 붙여넣기] 위치에 .note/*.md 전체 내용 붙여넣기
4. Claude가 HTML 코드 생성
5. 코드를 .pdf/output/{SUBJECT}_핵심요약노트.html 로 저장
6. 브라우저에서 열기
7. Ctrl+P → "PDF로 저장"
   ⚠️  반드시 "배경 그래픽(Background graphics)" 옵션 체크
```
