## 섹션 2: 토픽별 핵심 개념

### 1위. 동기이론 (24회 출제)

#### 데시와 라이언(Deci & Ryan)의 자기결정이론(SDT)

| 동기 유형 | 핵심 특징 | 기출 사례 |
|-----------|-----------|-----------|
| 무동기 | 행동에 가치 없음, 의욕 없음 | 외적동기 유형이 **아님** (2023년 함정) |
| 외적규제 | 보상·처벌 | 상금 받으려고 운동 |
| 의무감규제(내사규제) | 죄책감·수치심·타인 시선 | 부정적 평가받기 싫어 클럽 유지 (2019), 부모님 죄책감 (2021) |
| 확인규제 | 가치·중요성 스스로 인정 | 건강을 위해 조깅 (2021) |
| 통합규제 | 자아 통합, 가장 내면화된 외적동기 | 외적동기이지만 내적동기에 가장 가까움 (2023) |
| 내적동기 | 순수한 즐거움·흥미 | 스포츠 자체가 좋아서 (2024) |

**인지평가이론(CET)** — 데시와 라이언(Deci & Ryan)
내적동기를 높이는 3가지 기본심리욕구: **유능감(Competence) + 자율성(Autonomy) + 관계성(Relatedness)**
외적 보상의 강조 → 과잉정당화 효과 → 내적동기 저하 (2021)

#### 와이너(Weiner)의 귀인이론

| 귀인 요소 | 소재 | 안정성 | 통제가능성 |
|-----------|------|--------|-----------|
| 능력 | 내적 | 안정 | 불가능 |
| 노력 | 내적 | 불안정 | **가능** |
| 과제난이도 | 외적 | **안정** | 불가능 |
| 운(luck) | 외적 | 불안정 | 불가능 |

- 학습된 무기력 해결: 능력(안정·통제불가) → **노력·전략 미흡(불안정·통제가능)**으로 귀인 재훈련 (2020, 2023)
- 상대 선수의 능력: 외적·안정적·통제 불가능 (2019)
- 과제난이도는 외적·**안정적**·통제불가능 — "외적·불안정·통제가능"은 오답 (2022)

#### 하터(Harter)의 유능성 동기이론
성공 → 자기효능감·긍정적 정서 → 유능성 동기 상승 → 숙달 (2023)
단일차원이 아닌 **다차원적** 구성개념 — 단일차원이라는 선지는 오답 (2023)

### 2위. 운동학습·기억·정보처리 (23회 출제)

#### 피츠와 포스너(Fitts & Posner)의 3단계 운동학습

| 단계 | 특징 |
|------|------|
| 인지 단계 | 기술의 개념 이해, 무엇을 해야 하는지 파악 |
| 연합 단계 | 오류를 스스로 탐지·수정하며 연습 |
| 자동화 단계 | 안정적이고 일관성 있는 수행 |

#### 번스타인(Bernstein)의 운동학습 3단계
자유도 고정(관절 묶기) → 자유도 풀림(관절 독립 사용) → 반작용 활용(외부 힘·반동 이용) (2022)

#### 슈미트(Schmidt)의 도식이론(Schema Theory)

| 도식 | 역할 | 형성 요소 |
|------|------|-----------|
| 회상도식 | 빠른 운동 생성 (피드백 없이) | 초기조건 + 반응명세 + 실제결과 |
| 재인도식 | 느린 운동 오류 평가 | 초기조건 + 반응명세 + 감각귀결 |

- "재인도식이 빠른 운동을 조절한다" → **회상도식**이 맞음 (2022 함정)
- 일반화된 운동프로그램 (Generalized Motor Program: GMP)
  - GMP 불변매개변수: 요소의 순서·시상·상대적 힘
  - GMP 가변매개변수: 전체 지속시간·전체 힘·선택 근육군 (2022)
  - GMP 불변 특성의 대표 = **상대적 타이밍** (2023)

#### 정보처리 3단계

자극 확인 → 반응 선택 → 반응실행(운동 프로그래밍)

- 반응시간(RT) = 자극 제시 ~ 반응 시작
- 움직임 시간(MT) = 반응 시작 ~ 반응 종료
- 전체 반응시간 = RT + MT (2020)

**힉스의 법칙(Hick's Law)**: 자극-반응 대안 수가 늘수록 반응시간이 **로그함수적**으로 증가 (2022, 2024, 2025)

**반응시간 3유형**:
- 단순반응시간: 1자극 → 1반응 (100m 출발신호)
- 변별반응시간: 다자극 중 특정 자극에만 반응 (직구에만 타격)
- 선택반응시간: 다자극 → 각각 다른 반응 (수비에 따라 대응) (2019, 2025)

**심리적 불응기(PRP)**: 1차 자극 반응 중 2차 자극이 제시되면 2차 반응이 지연 — 페이크 동작의 원리 (2022, 2024)

#### 기억 체계

| 기억 | 용량 | 특징 |
|------|------|------|
| 감각기억 | 극히 제한 | 새 정보 유입 시 즉각 손실 |
| 단기기억 | 제한적(7±2 청크) | 시연 없으면 사라짐 |
| 장기기억 | 무제한 | 반복으로 강화 |

**절차적 기억**: '어떻게 하는지' 운동기술 기억 — 자전거 타기, 슛 기술 (2024)

#### 연습 방법 정리

| 구분 | 의미 |
|------|------|
| 구획연습(blocked) | 같은 기술 집중 반복 → 단기 수행 유리 |
| 무선연습(random) | 기술 무작위 배열 → 파지·전이 유리 (맥락간섭 효과) |
| 계열연습(serial) | 일정 순서 반복 → 중간 수준 |
| 집중연습(massed) | 휴식 < 연습 시간 |
| 분산연습(distributed) | 휴식 ≥ 연습 시간 |
| 전습법 | 과제 전체를 분리 없이 연습 |
| 분절화(segmentation) | 시간적 순서로 부분 분리 후 통합 (순수·점진적 분습법) |
| 부분화(fractionization) | 동시 수행 요소를 분리 연습 |

맥락간섭 높음: 무선연습 > 계열연습 > 구획연습 (2020, 2024, 2025)

#### 전이 검사 vs 파지 검사

- **파지검사**: 일정 시간 후 동일 과제 재수행 (기억 유지 평가) — 수영장 → 24시간 후 동일 과제 (2019, 2020, 2023)
- **전이검사**: 새로운 상황·과제에서의 수행 — 수영장 → 바다 (2019, 2020)

#### 피드백 유형

- **보강적 피드백**: 외부에서 추가 제공 (KR, KP)
  - KR (결과지식): 결과·점수 정보
  - KP (수행지식): 동작 패턴·운동 형태 정보 — "백스윙이 안 됐어" (2020)
- **내재적 피드백**: 감각기관으로 스스로 감지 (시각·촉각·청각)

### 3위. 각성·불안 이론 (14회 출제)

| 이론 | 학자 | 핵심 내용 |
|------|------|-----------|
| 추동이론(Drive Theory) | Hull & Spence | 각성과 수행은 선형 비례 (2025) |
| 역U자 가설 | Yerkes & Dodson | 중간 각성에서 최고 수행 |
| 다차원불안이론 | Martens 등 | 인지불안(선형 부적) + 신체불안(역U) + 자신감 구분 |
| 반전이론(Reversal Theory) | Apter | 동일 각성을 흥분(+) 또는 불안(-)로 해석 → 전환 (2019, 2022) |
| 격변이론(Catastrophe Theory) | Hardy | 인지불안 높을 때 각성↑→ 수행 급격히 추락 |
| 최적수행지역이론(ZOF) | Hanin | 개인마다 최적 불안 구간이 다름 (2019, 2021) |

**상태불안 vs 특성불안**:
- 상태불안: 특정 상황에서 일시적으로 경험 — 앞 선수의 뛰어난 연기를 보고 긴장 (2020)
- 특성불안: 개인의 기질적·성격적 경향성, 지속적

**측정 도구**:
- SCAT: 스포츠 경쟁 **특성불안** 측정 (마텐스)
- CSAI-2: 경쟁 **상태불안** 측정 (인지적·신체적·자신감 3요인) (2021)

**지각 협소화(perceptual narrowing)**: 각성 상승 시 주의 폭이 좁아지는 현상 (2020, 2022)

### 4위. 심리기술훈련 (14회 출제)

#### 이완 기법 비교

| 기법 | 개발자 | 핵심 방법 |
|------|--------|-----------|
| 점진적 이완(Progressive Relaxation) | 제이콥슨(Jacobson) | 근육 긴장→이완 반복 |
| 자생훈련(Autogenic Training) | 슐츠(Schultz) | 자기암시(온감·중량감) |
| 바이오피드백(Biofeedback) | — | 생체신호를 시각·청각으로 변환해 자기조절 (2019) |
| 체계적 둔감화(Systematic Desensitization) | 월피(Wolpe, 1958) | 불안 위계 작성 후 점진적 노출·이완 (2022, 2025) |
| 인지 재구성(Cognitive Restructuring) | — | 부정적 사고 → 합리적·긍정적 사고로 전환 (2020, 2024) |

#### 루틴(Routine)

- 불안 감소·집중력 증대·편안함 유지에 효과
- **시합 당일(경기 직전)에 수정하면 안 됨** — 매년 반복 오답 선지 (2019, 2025)
- 심상과 혼잣말(자기대화)이 포함될 수 있음

#### 심리기술훈련(PST) 일반 원칙

- 즉각적 효과가 아닌 **지속적·체계적 연습**으로 효과 (2021)
- 연령·성별·경기수준과 관계없이 모든 선수에게 적용 가능

### 5위. 집단역학·리더십 (12회 출제)

#### 캐런(Carron)의 팀 응집력 4대 결정요인

환경 요인 + 개인 요인 + 리더십 요인 + **팀 요인**
- **"발달 요인"은 존재하지 않음** — 함정 선지로 반복 출제 (2021)
- 팀의 규모 = **환경 요인** (팀 요인이 아님) (2019)

#### 링겔만 효과와 사회적 태만

- 링겔만 효과: 집단 총 수행력 < 개인 수행력 합산
- 원인: **동기 손실(사회적 태만)** — "유능감 손실"은 오답 (2020, 2025)
- 극복 전략: 소집단 구성, 개인 기여 식별, 개별 책임 부여 (2021)

#### 첼라드라이(Chelladurai)의 다차원리더십 모델

| 행동 유형 | 정의 |
|-----------|------|
| 규정 행동 | 상황·조직이 리더에게 요구하는 행동 |
| 실제 행동 | 리더가 실제로 보이는 행동 |
| 선호 행동 | **선수가** 리더에게 원하는 행동 |

"규정 행동 = 선수에게 규정된 행동" → 오답 (2022 함정)
"선호 행동 = 리더가 선호하는 선수의 행동" → 오답 (2022 함정)

#### 피들러(Fiedler)의 상황부합 리더십

- 상황 호의성 매우 높거나 낮을 때 → **과제지향 리더** 효과적
- 상황 호의성 중간일 때 → **관계지향 리더** 효과적 (2024)

#### 사회적 촉진 3가설

- 단순존재가설(Zajonc): 타인 존재만으로 각성↑
- 평가우려가설(Cottrell): 평가받는다는 우려 때문에 각성↑
- 주의 분산·갈등 가설(Baron): 주의 분산하지만 수행 욕구도↑ (2024)

### 6위. 주의집중·심상 (11회 출제)

#### 나이데퍼(Nideffer)의 주의초점모형

| | 외적(External) | 내적(Internal) |
|--|----------------|----------------|
| **광의(Broad)** | 환경 전체 파악 (서브 전 상대 진영 파악) | 전술·계획 분석 |
| **협의(Narrow)** | 특정 외부 목표 집중 (빈 곳으로 서브) | 특정 신체 감각 집중 |

배구 서브 사례: 광의 외적 → 협의 외적으로 전환 (2021)

#### 심상의 2대 요소

- **선명도(Vividness)**: 실제처럼 생생하게 감각 재현
- **조절력(Controllability)**: 심상 내용을 원하는 방향으로 변경·수정
  - "부정적 장면을 성공 이미지로 바꾼다" = 조절력 (2020)
  - "복싱 선수가 실신 장면이 계속 떠오른다" → 심상 조절력 훈련 필요 (2024)

#### 심상 이론 3가지

| 이론 | 핵심 내용 |
|------|-----------|
| 심리신경근 이론 | 심상 시 실제 근육에서 미세 전기 반응 발생 |
| 상징학습 이론(Sackett) | **인지 과제(바둑)**에서 더 효과적 — 운동 과제보다 인지 과제 (함정!) |
| 생물정보 이론(Lang) | 자극 전제(stimulus proposition) + 반응 전제(response proposition) |

#### 주의집중 방해 요소

- 해저드를 생각하는 것, 과거 실수를 떠올리는 것 → 주의집중 방해 (2020, 2025)
- 칵테일파티 효과: 시끄러운 환경에서도 의미 있는 정보에 선택적 주의 (2023)

### 7위. 운동발달 (9회 출제)

#### 갤라휴(Gallahue)의 7단계 운동발달

반사운동단계 → 초기움직임단계 → 기본움직임단계 → **스포츠기술단계** → **성장과 세련단계** → 최고수행단계 → 퇴보단계 (2021, 2024)

- 스포츠기술단계(7~10세)가 성장과 세련단계(11~13세) **앞에** 옴 — 순서 바꾸면 오답

#### 게셀과 에임스(Gesell & Ames)의 발달 원리

- 머리-꼬리 원리(cephalocaudal): 머리에서 발 방향
- 중앙-말초 원리(proximodistal): 몸통·어깨 → 팔 → 손목·손가락
- 양측-동측-교차 운동협응 원리
- **개체발생적 발달 원리는 게셀·에임스의 원리가 아님** (2023 함정)

#### 개체발생적 vs 계통발생적 운동행동

- **계통발생적**: 인류 공통·보편 (걷기, 달리기) — "공통적으로 나타나는 행동"
- **개체발생적**: 경험·학습에 의해 개인마다 다름 (수영, 자전거)
- "공통적으로 나타나는 행동 = 개체발생적" → 오답 (2021 함정)

### 8위. 운동행동·건강심리 (9회 출제)

#### 프로차스카(Prochaska)의 변화단계이론(TTM)

| 단계 | 정의 | 전략 |
|------|------|------|
| 무관심(전고려) | 6개월 내 운동 의도 없음 | 긍정적 정보 제공·인식 높이기 (2025) |
| 고려(관심) | **6개월 내** 운동 의도 있음 | 이점·장벽 분석 |
| 준비 | **1개월 내** 의도 + 일부 실천 | 구체적 계획 수립 |
| 실행(행동) | 6개월 **미만** 규칙적 운동 | 목표 설정·지지 |
| 유지 | 6개월 **이상** 규칙적 운동 | 재발 방지·자기효능감 |

**함정**: "준비단계 = 6개월 이내에 운동 시작 의도" → **관심(고려) 단계**의 설명 (2020 오답)
자기효능감: 유지 단계에서 가장 높음
변화단계가 높을수록 운동 혜택 인식(pros)↑, 장애 인식(cons)↓

#### 아이젠(Ajzen)의 계획된 행동이론(TPB)

**태도 + 주관적 규범 + 행동통제인식 → 의도 → 행동**
- 행동통제인식: 행동을 수행하기가 쉽거나 어렵다는 인식 (자기효능감 개념 통합)
- "동기(motivation), 자신감(confidence)"은 TPB 구성요소가 **아님** (2021 함정)

#### 건강심리 가설 (2025 신출제)

| 가설 | 핵심 |
|------|------|
| 모노아민 가설 | 세로토닌·노르에피네프린·도파민 분비↑ → 우울증 개선 |
| 열발생 가설 | 체온 상승 → 근육 이완·불안 감소 |
| 사회심리적 가설 | 사회적 상호작용·자기효능감으로 정신건강 향상 |
| 생리적 강인함 가설 | 반복 운동으로 스트레스 반응계 강화 |

### 9위. 목표설정 (8회 출제)

#### 목표 유형 3가지

| 유형 | 기준 | 특징 |
|------|------|------|
| 과정목표 | 핵심 행동·기술 수행 방법 | 자기효능감·자신감↑, 인지불안↓ |
| 수행목표 | 자신의 **과거 기록** | 통제 가능, 개인 향상 중심 |
| 결과목표 | 승패·순위·타인 비교 | 외부 변수 의존, 통제 불가 |

**원칙**: 수행목표·과정목표 강조 — 결과목표를 강조하는 것은 **올바르지 않은** 목표설정 원리 (2025)

로크와 라탐(Locke & Latham): 어렵고 구체적인 목표가 쉬운 목표보다 수행 향상에 효과적

### 10위. 상담 윤리·측정 (8회 출제)

#### 핵심 윤리 원칙

- **이중 관계 금지**: 가족·친구·알고 지내는 사람과 전문 상담 **금지** (2019, 2025)
- **비밀보장**: 내담자 동의 없이 상담 내용 공유 금지
- 고객으로부터 좋은 평가·소감을 **요구하지 않음** (2022)
- 역할 위임 시 전문성 있는 사람에게만 위임

#### AAASP 윤리 규정

"스포츠에 참여하는 모든 사람과 전문적인 상담을 진행한다" → **오답** (역량 범위 무시) (2024)

#### 측정 도구

| 도구 | 측정 내용 |
|------|-----------|
| SCAT (마텐스) | 스포츠 경쟁 **특성불안** |
| CSAI-2 (마텐스 등) | 경쟁 **상태불안** (인지적·신체적·자신감) |
| ABQ (Raedeke & Smith) | 선수 탈진 (성취감 저하·평가절하·고갈) |
| POMS | 기분 상태 (6척도: 긴장·우울·분노·활기·피로·혼란) |
| TAT (주제통각검사) | **투사검사** — 질문지법이 **아님** (2025) |

### 11위. 성격·성취목표이론 (6회 출제)

#### 성격 이론 4가지

| 이론 | 핵심 |
|------|------|
| 특성이론 | 유전적으로 타고난 안정적 특성 |
| 사회학습이론(Bandura) | 관찰·모방으로 행동·성격 형성 (2024) |
| 정신역동이론(Freud) | 무의식·충동·방어기제 |
| 욕구위계이론(Maslow) | 생리→안전→소속→존중→자아실현 |

#### 성취목표이론

| 목표성향 | 특징 |
|----------|------|
| 과제목표성향(task) | 기술 향상·노력 중시, 자기 참조적, 실패 시 노력 귀인 |
| 자기목표성향(ego) | 타인 대비 우월, 사회 비교 중심, 승리 시 능력 귀인 |

과제목표성향: 실패 시 **노력 부족**으로 귀인 (2019)
자기목표성향: 실패 시 상대 실수나 외적 요인으로 귀인, 평가 상황에서 수행 저조

#### 폭스(Fox)의 위계적 신체적 자기개념

전반적 자기존중감(상위) → 신체적 자기가치(하위) → 스포츠 유능감·신체적 컨디션·신체 매력·체력
- "신체적 자기가치가 전반적 자기존중감의 **상위**영역" → **오답** (하위 영역임) (2022 함정)
- 신체적 컨디션 = 체력·운동능력 유지 (매력적 신체 유지가 아님)

### 12위. 스포츠심리학 역사·개념 (4회 출제)

"생리학적 항상성 연구"는 스포츠심리학 영역이 **아님** (운동생리학) (2023)

## 섹션 3: 함정 선지 총정리

❌ "타인의 수행에 대한 관찰은 자기효능감에 영향을 주지 않는다" → **대리 경험**은 자기효능감 4대 정보원 중 하나 (2019)

❌ "준비단계는 현재 운동에 참여하지 않지만 6개월 이내에 운동을 시작할 의도가 있다" → **관심(고려) 단계**의 설명, 준비단계는 1개월 이내 + 소량 실천 (2020)

❌ "재인도식은 피드백 정보가 없는 빠른 운동을 조절한다" → **회상도식**이 빠른 운동 생성, 재인도식은 느린 운동 오류 평가 (2022)

❌ "회상도식은 과거의 실제결과·감각귀결·초기조건으로 형성된다" → 회상도식은 **초기조건·반응명세·실제결과**, 감각귀결은 **재인도식** 형성에 사용 (2022)

❌ "규정 행동은 선수에게 규정된 행동이다" → 규정 행동은 **상황·조직이 리더에게 요구**하는 행동 (2022)

❌ "선호 행동은 리더가 선호하는 선수의 행동이다" → 선호 행동은 **선수가 리더에게 원하는** 행동 (2022)

❌ "링겔만 효과의 원인은 유능감 손실이다" → 원인은 **동기 손실(사회적 태만)** (2020)

❌ "캐런 응집력 모형에 발달 요인이 있다" → **발달 요인 없음**, 4요인은 환경·개인·리더십·팀 (2021)

❌ "과제목표성향 선수는 달성 불가능한 과제를 자주 선택한다" → 과제목표성향은 현실적·도전적 목표 선택 (2021)

❌ "공통적으로 나타나는 행동 = 개체발생적 운동행동" → **계통발생적** 운동행동 (2021)

❌ "신체적 자기가치는 전반적 자기존중감의 상위영역이다" → **하위영역** (2022)

❌ "분산연습은 과제를 여러 하위 단위로 나누어 연습하는 방법이다" → **분습법(part practice)**의 설명, 분산연습은 연습 간 충분한 휴식을 두는 것 (2024)

❌ "상징학습 이론에 따르면 심상은 인지 과제보다 운동 과제에서 더 효과적이다" → 반대, **인지 과제(바둑 등)**에서 더 효과적 (2022)

❌ "상징학습 이론에 따르면 생리·심리 반응을 함께하면 심상 효과가 낮아진다" → **생물정보 이론**의 내용, 반응 전제 활성화가 높을수록 효과↑ (2022)

❌ "심리기술훈련은 경기력 향상에 즉각적 효과를 준다" → 즉각적 효과 아닌 **지속적·체계적 연습** 후 효과 (2021)

❌ "과제난이도는 외적이고 불안정하며 통제가능한 요인이다" → 외적·**안정적**·통제**불가능** (2022)

❌ "루틴은 시합 당일에 수정한다" → 루틴은 일관성이 핵심, 시합 당일 수정 금지 (2019, 2025)

❌ "무관심 단계에서 다른 사람의 운동 멘토 역할을 하도록 한다" → 멘토 역할은 **유지 단계** 전략 (2025)

❌ "스포츠에 참여하는 모든 사람과 전문적인 상담을 진행한다" → 역량 범위 내에서만 상담, 이중 관계 금지 (2024)

❌ "자기목표성향(ego-goal orientation) 선수는 자신의 노력 부족으로 패배했다고 생각한다" → 노력 귀인은 **과제목표성향**의 특징 (2019)

❌ "변화단계와 자기효능감의 관계는 U자 형태다" → 단계가 높아질수록 자기효능감 **단조 증가** (2022)

## 섹션 4: 최근 3년 신출제 경향 (2023~2025)

### 2023년 — 이론 적용·분류 심화

- **뉴웰(Newell)의 제한 이론** 신출제: 유기체 제한 / 과제 제한 / 환경 제한 3분류
- **젠타일(Gentile)의 이차원 운동기술 분류** (표 해석형): 환경 맥락 × 행동 기능
- **GMP 불변 특성 = 상대적 타이밍** (어트랙터·동작유도성과 구별)
- 공격성 이론 4가지 연결: 사회학습·본능·좌절-공격·수정된 좌절-공격 가설
- 칙센트미하이(Csikszentmihalyi) 몰입 모델: 기술·도전의 균형, 불안·이완 방향
- 구스리(Guthrie)의 운동기술 학습 변화 3요소: 최대 확실성·최소 움직임 시간·최소 에너지 소비

### 2024년 — 학자명·이론 매칭 강화

- **피들러(Fiedler)의 상황부합 리더십** 신출제 (그래프 해석)
- **절차적 기억(procedural memory)** vs 의미·일화·감각 기억 구별
- **사회적 촉진 3가설** 정밀 구분 (단순존재·평가우려·주의분산)
- **천장 효과·바닥 효과** 신출제 (과제 난이도와 수행 측정)
- **개방회로이론·폐쇄회로이론·도식이론** 3가지 동시 비교
- 사회생태모형 신출제: 개인·환경·정책 다층 개입

### 2025년 — 건강심리·역사 강화

- **모노아민 가설** 신출제: 세로토닌·노르에피네프린·도파민과 우울증
- **콜먼 그리피스(Coleman Griffith)** 신출제: 북미 스포츠심리학의 아버지
- **베일리(Vealey)의 스포츠자신감 원천** 신출제: 성취경험·사회적 분위기·자기조절
- **고원현상(plateau) 심화**: 협응 구조 재조직화 과정
- **본능이론·좌절-공격·수정된 좌절-공격·사회학습이론** 구별 (2023 공격성 이론 재출제)
- 변화단계이론 단계별 전략 정밀화

**트렌드 요약**: 단순 암기에서 이론가-이론-적용 상황의 3단 연결로 난이도 상승. 2025년은 건강심리(우울증 가설)·스포츠심리학 역사·자신감 이론이 새롭게 진입. 공격성 이론과 집단역학은 격년 반복 출제 경향.

인지(이해) → 연합(오류 수정) → 자동화(일관성)

[GMP 불변 vs 가변]
