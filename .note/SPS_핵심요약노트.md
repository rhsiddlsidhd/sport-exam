## 섹션 2: 토픽별 핵심 개념

### 1. 운동학습 — 피드백·연습구조·전이·파지

#### 피츠&포스너(Fitts & Posner) 운동학습 3단계

| 단계 | 특징 | 사례 |
|------|------|------|
| **인지 단계** | 과제 개념 이해, 인지적 노력 높음 | "포핸드 스트로크가 뭔지 이해한다" |
| **연합 단계** | 오류 탐지·수정 가능, 세련화 | "오류를 스스로 찾아 수정할 수 있다" |
| **자동화 단계** | 무의식적·안정적·일관된 수행 | "안정적이고 일관성 있게 수행한다" |

#### 번스타인(Bernstein) 자유도 학습 3단계

| 단계 | 특징 | 스케이트 예시 |
|------|------|--------------|
| **자유도 고정** | 관절 묶어 하나처럼 → 초보 | 고관절/슬관절/발목을 한 단위로 굳게 걷기 |
| **자유도 풀림** | 각 관절 활용 → 중간 | 세 관절 각각 사용해 추진력 확보 |
| **반작용 활용** | 외력(지면반발력) 이용 → 숙련 | 체중이동으로 반발력 확보 |

#### 보강적 피드백 유형

| 유형 | 정의 | 예시 |
|------|------|------|
| **KR** (결과지식) | 동작의 결과(성공/실패) | "파울이야", "10점이야" |
| **KP** (수행지식) | 동작의 질·형태·과정 | "백스윙이 제대로 안 됐어", "무릎을 더 구부려" |

> 내재적 피드백(시각·청각·촉각) ≠ 보강적 피드백 — 시각/청각은 내재적, KR/KP가 보강적

#### 연습 구조 비교

| 구분 | 특징 | 장기학습 |
|------|------|---------|
| **구획연습** (blocked) | AAABBBCCC → 맥락간섭 낮음 | 초보자에 유리, 즉각 수행 ↑ |
| **계열연습** (serial) | ABCABCABC → 맥락간섭 중간 | 중간 |
| **무선연습** (random) | ABCACBBA → 맥락간섭 높음 | 장기 보유·전이에 유리 |
| **집중연습** (massed) | 휴식 < 연습시간 | 피로 축적 |
| **분산연습** (distributed) | 휴식 충분 | 학습 효과 좋음 |

> **분절화(Segmentation)**: 기술을 시간 순서로 하위 단위로 나눔 (순수 분습법, 점진적 분습법 포함)
> **분습법(부분화, Fractionization)**: 동시에 움직이는 다른 신체부위를 분리 연습

#### 파지·전이 검사

| 검사 | 목적 | 설명 |
|------|------|------|
| **파지검사** (retention test) | 학습의 지속성 | 시간 경과 후 동일 과제 재수행 |
| **전이검사** (transfer test) | 새 환경·과제 적응력 | 다른 환경·유사 과제에 적용 |

- **정적 전이**: 이전 기술이 새 기술에 도움 (배구 → 테니스 점프서브)
- **부적 전이**: 이전 기술이 방해 (배드민턴 손목스냅 → 테니스 손목고정)

#### GMP(일반화된 운동프로그램) 슈미트(Schmidt)

| 매개변수 | 구성 요소 |
|----------|----------|
| **불변(invariant)** | 요소의 순서, 상대적 타이밍(시상), 상대적 힘 |
| **가변(variant)** | 전체 동작 지속시간, 힘의 총량, 선택된 근육군 |

> 상대적 타이밍: 전체 시간(절대적 타이밍)이 바뀌어도 하위 구간 비율은 불변 → GMP 핵심 증거

### 2. 동기이론

#### 자기결정이론(SDT) — 데시&라이언(Deci & Ryan)

```
무동기 ← 외적규제 ← 의무감규제(introjected) ← 확인규제(identified) ← 통합규제(integrated) ← 내적동기
```

| 유형 | 핵심 | 사례 |
|------|------|------|
| **무동기** | 의지 없음 | 외적동기 아님 — 별도 범주 |
| **외적규제** | 상/벌 | "상금 받으려고" |
| **의무감규제** | 죄책감·수치심 | "그만두면 부모님께 죄책감이 들어서" |
| **확인규제** | 가치 인정·스스로 선택 | "건강을 위해 중요하다고 판단해서" |
| **내적동기** | 순수 즐거움 | "배드민턴이 너무 재미있어서" |

- 내적동기 향상 욕구: **자율성 + 유능감 + 관계성**
- **인지평가이론**: 외적 보상 지나치게 강조 → **과잉정당화 효과** → 내적동기 저하

#### 귀인이론(Attribution Theory) — 와이너(Weiner)

| 귀인 원인 | 인과소재 | 안정성 | 통제가능성 |
|----------|---------|--------|-----------|
| **능력** | 내적 | 안정 | 불가 |
| **노력** | 내적 | **불안정** | **가능** ← 귀인재훈련 목표 |
| **운** | 외적 | 불안정 | 불가 |
| **과제난이도** | 외적 | **안정** | 불가 |

> 학습된 무기력: 실패 → 능력(내/안/불가)으로 귀인 → 재훈련으로 노력(내/불안/가능)으로 전환

#### 성취목표성향이론

| 성향 | 특징 | 실패 귀인 |
|------|------|----------|
| **과제(mastery)목표** | 자신의 성장·숙달·노력 | 노력 부족 |
| **자기(ego)목표** | 타인 비교·승리·능력 과시 | 타인 우월 또는 운 탓 |

#### 목표설정

- **결과목표** (승패, 순위) → 통제 낮음
- **수행목표** (자신의 과거 기준) → 통제 중간
- **과정목표** (핵심 동작·행동에 집중) → 통제 높음, 자신감↑ 불안↓

> 바람직한 원리: **수행/과정목표 강조** > 결과목표, 구체적, 긍정적, 단기·중기·장기 병행

### 3. 사회심리

#### 캐런(Carron) 팀 응집력 결정요인 4가지

| 요인 | 내용 |
|------|------|
| **환경 요인** | 계약, 지리적 조건, 팀 규모 |
| **개인 요인** | 개인적 특성, 동기, 만족감 |
| **리더십 요인** | 리더의 행동과 스타일 |
| **팀 요인** | 팀 목표, 공유 규범, 팀 능력 |

> ❗ 팀 규모 → 환경 요인 (팀 요인 아님)

#### 사회적 태만과 링겔만 효과

- **링겔만 효과**: 집단 커질수록 1인당 기여도 감소
- **원인**: 동기 손실 (유능감 손실 X)
- **극복 전략**: 개인 수행 식별 가능하게 하기, 소집단 구성, 역할 명확화, 개인 노력 인정

#### 리더십 이론

| 이론 | 학자 | 핵심 |
|------|------|------|
| **상황부합 이론** | 피들러(Fiedler) | 고/저통제 → 과제지향, 중통제 → 관계지향 |
| **다차원 리더십 모델** | 첼라드라이(Chelladurai) | 규정행동(조직→리더), 선호행동(선수→리더), 실제행동 일치 → 만족 |

> 규정 행동 = 조직이 리더에게 요구, 선호 행동 = 선수(구성원)가 리더에게 원하는 것

#### 공격성 이론

| 이론 | 핵심 |
|------|------|
| **본능이론** | 내재된 공격 에너지 → 자연 표출 |
| **사회학습이론** | 관찰·강화로 학습 (반두라) |
| **좌절-공격 가설** | 좌절 → 무조건 공격 |
| **수정된 좌절-공격 가설** | 좌절 + 공격 단서(cue) → 공격 |

### 4. 운동제어/정보처리

#### 정보처리 3단계

```
자극 확인 → 반응 선택 → 반응 실행(운동 프로그래밍)
```

#### 반응시간 3유형

| 유형 | 구조 | 스포츠 사례 |
|------|------|------------|
| **단순반응시간** | 자극1 → 반응1 | 100m 출발 총소리 |
| **선택반응시간** | 자극N → 반응N | 수비수 움직임에 따라 다르게 패스 |
| **변별반응시간** | 자극N → 특정 자극에만 반응 | 직구에만 타격 |

#### 시간 지표

| 용어 | 정의 |
|------|------|
| **반응시간(RT)** | 자극 제시 ~ 반응 시작 |
| **움직임 시간(MT)** | 반응 시작 ~ 반응 종료 |
| **전체 반응시간** | RT + MT |

#### 주요 법칙

| 법칙 | 학자 | 내용 |
|------|------|------|
| **힉스 법칙** | 힉(Hick) | 자극-반응 대안 수↑ → **반응시간↑** (로그함수) |
| **피츠 법칙** | 피츠(Fitts) | 목표 거리↑ or 크기↓ → **움직임 시간↑** |
| **심리적불응기(PRP)** | — | 1차 처리 중 2차 자극 → 2차 반응 지연 (페이크 원리) |

### 5. 심리기술훈련

#### 심상(Imagery)

| 요소 | 정의 | 사례 |
|------|------|------|
| **선명도(vividness)** | 모든 감각으로 생생하게 | 시합의 흥분감, 함성 소리까지 |
| **조절력(controllability)** | 원하는 방향으로 바꾸는 능력 | 실점 장면 → 성공 장면으로 전환 |

- **내적 심상**: 자신의 눈에서 본 시점 / **외적 심상**: 제3자 시점에서 관찰
- **심상의 기능**: 동기 유발, 감정 조절, 전략 습득, 부상 대처·재활 의지 향상

#### 심상 이론

| 이론 | 핵심 |
|------|------|
| **심리신경근 이론** | 심상 시 실제 근육에 유사 전기 신호 발생 |
| **상징학습 이론** | 인지 과제(바둑 등)에서 더 효과적 |
| **생물정보 이론** | 심상 = 자극 전제 + 반응 전제 조합 (Lang) |

#### 이완기법 비교

| 기법 | 핵심 원리 | 특징 |
|------|----------|------|
| **바이오피드백** | 생체신호(심박·근전도)를 시각/청각으로 변환 | 무의식적 반응을 의식적으로 조절 |
| **자생훈련** | "팔이 무겁다, 따뜻하다" 자기최면 유도 | 슐츠(Schultz) 개발 |
| **점진적 이완** | 근육군 순서대로 수축-이완 반복 | 제이콥슨(Jacobson) |
| **체계적 둔감화** | 불안 계층 목록 → 낮은 것부터 이완과 함께 | 월피(Wolpe, 1958) 개발 |

#### 루틴(Routine)

- 시합 당일 수정 **금지** (일관성 깨짐)
- 구성 요소: 심상, 혼잣말(self-talk)
- 기능: 불안 감소, 집중력 증대, 상황 변화에서도 편안함 유지

### 6. 스포츠심리학 기초·상담윤리

#### 분야 구분

| 분야 | 핵심 질문 |
|------|----------|
| **운동제어** | 동작을 어떻게 생성·조절하는가 |
| **운동학습** | 연습으로 어떻게 기술이 변화하는가 |
| **운동발달** | 전 생애에 걸쳐 어떻게 발달하는가 |
| **스포츠심리학** | 심리가 수행에, 수행이 심리에 어떤 영향을 미치는가 |

#### 스포츠심리학 역사

- **그리피스(Coleman Griffith)**: 북미 스포츠심리학의 아버지, 최초 실험실 설립(1925), 시카고 컵스 상담, 『코칭심리학(1926)』 저술

#### 상담 윤리 핵심 (AAASP·한국스포츠심리학회 공통)

| 원칙 | 내용 |
|------|------|
| **이중관계 금지** | 가족·친구·지인과 상담 X, 사적 관계 유지 X |
| **비밀보장** | 내담자 동의 없이 타인 공유 X |
| **전문성 한계** | 범위 밖 상담 X → 적절한 전문가에게 의뢰 |
| **자율성 존중** | 긍정적 평가·소감 강요 X |

#### 측정 도구

| 도구 | 측정 내용 |
|------|----------|
| **SCAT** | 특성불안 |
| **CSAI-2** | 상태불안 (경쟁상태불안 3요소) |
| **POMS** | 기분상태 |
| **MBTI / 16PF** | 성격 (질문지법) |
| **TAT** (주제통각검사) | 투사적 기법 — **질문지 아님** |

### 7. 운동행동이론

#### 변화단계이론(TTM) — 프로차스카(Prochaska)

| 단계 | 특징 |
|------|------|
| **무관심** | 6개월 내 운동 의사 없음 → 긍정적 효과 정보 제공 |
| **관심(숙고)** | 6개월 내 시작 의도 |
| **준비** | 1개월 내 시작 or 불규칙적 운동 중 |
| **실천** | 규칙적 운동 6개월 미만 |
| **유지** | 규칙적 운동 6개월 이상 |

- 변화단계 높을수록 **자기효능감 ↑**, 운동 혜택 인식 ↑
- **의사결정균형**: 혜택 vs 손실 평가
- 변화 과정: 인지적 과정 + 행동적 과정

#### 계획행동이론(TPB) — 아이젠(Ajzen)

```
태도 + 주관적 규범 + 행동통제인식(PBC) → 의도 → 행동
```

- **태도**: 행동에 대한 정서적·평가적 요소
- **주관적 규범**: 사회적 압력 (타인의 기대)
- **행동통제인식**: 행동 쉽고 어려운 정도에 대한 인식 (Perceived Behavioral Control)

#### 사회생태모형

- 개인 의지뿐 아니라 **가정·학교·지역사회·정책** 등 다층 환경 체계가 운동 행동에 영향

---

### 8. 자기개념/자신감

#### 자기효능감(Self-Efficacy) — 반두라(Bandura)

| 정보원 | 영향력 순위 | 예시 |
|--------|-----------|------|
| **성취경험** | **1위 (가장 강력)** | 직접 성공 경험 |
| **대리경험** | 2위 | 타인의 성공 관찰 |
| **언어적 설득** | 3위 | 코치의 격려 |
| **신체적·정서적 상태** | 4위 | 각성 상태 해석 |

#### 스포츠자신감 — 베일리(Vealey)

| 원천 | 사례 |
|------|------|
| **성취 경험** | "시합에서 좋은 성과를 낸다" |
| **사회적 분위기** | "주변에서 나를 믿어준다" |
| **자기조절** | "체력·전략·정신력을 갖춘다" |

#### 번아웃(Burnout) — 레이데크&스미스 ABQ 3요인

1. 성취감 저하
2. 스포츠 평가절하
3. 신체적·정서적 고갈

#### 몰입(Flow) — 칙센트미하이(Csikszentmihalyi)

| 조건 | 결과 |
|------|------|
| 기술 = 도전 (균형) | **몰입** |
| 도전 ↑ > 기술 | 불안 |
| 기술 ↑ > 도전 | 이완·지루함 |

#### 폭스(Fox) 위계적 신체자기개념

```
자기존중감 > 신체적 자기가치 > [스포츠유능감 / 신체매력 / 신체컨디션 / 신체근력]
```

### 9. 운동발달

#### 갈라휴(Gallahue) 운동발달 7단계

```
반사 → 초기움직임 → 기본움직임 → 스포츠기술 → 성장과 세련 → 최고수행 → 퇴보
```

#### 게셀(Gesell)&에임스(Ames) 발달 원리

| 원리 | 방향 |
|------|------|
| **머리-꼬리(cephalocaudal)** | 머리 → 발 방향 |
| **중앙-말초(proximodistal)** | 몸통/어깨 → 팔 → 손목 → 손 → 손가락 |
| **양측 → 동측 → 교차** 협응 순서 |

#### 뉴웰(Newell) 움직임 제한 3요소

| 요인 | 내용 |
|------|------|
| **개인(유기체)** | 운동능력, 인지·동기·정서, 신장·체중 |
| **과제** | 목표, 규칙, 장비 ← 유일하게 다른 범주 |
| **환경** | 중력, 온도, 빛 |

### 10. 불안이론

#### 이론 비교

| 이론 | 핵심 |
|------|------|
| **추동이론** (Drive) | 각성↑ → 수행↑ (선형, 비례) |
| **역U자 가설** | 최적 각성 수준 존재 |
| **다차원불안이론** | 인지불안(-)과 신체불안이 수행에 다르게 영향 |
| **반전이론** (Reversal) | 동일 각성도 해석에 따라 흥분↔불안으로 전환 |
| **IZOF** (한닌, Hanin) | 선수마다 고유한 최적 불안 범위 존재 |
| **카타스트로피이론** | 인지불안 높을 때 신체각성 임계점 넘으면 급격한 수행 붕괴 |

#### 상태불안 vs 특성불안

| 구분 | 특징 | 측정도구 |
|------|------|---------|
| **상태불안** | 특정 상황·일시적 | CSAI-2 |
| **특성불안** | 성격적·안정적 성향 | **SCAT** |

## 섹션 3: 함정 선지 총정리

❌ **"타인의 수행 관찰은 자기효능감에 영향을 주지 않는다"**
→ 대리경험(간접경험)은 4대 정보원 중 2위로 중요한 원천

❌ **"루틴은 시합 당일에 수정한다"**
→ 당일 수정 금지. 충분히 자동화된 후 사용해야 효과 있음

❌ **"과제난이도는 외적·불안정·통제 가능한 요인"**
→ 외적·**안정적**·통제 **불가능** (과제난이도는 쉽게 안 바뀜)

❌ **"상징학습이론은 운동과제에서 더 효과적"**
→ 상징학습이론은 동작 패턴 부호화를 강조하므로 **인지 과제(바둑 등)에서 더 효과적**

❌ **"재인도식은 피드백 없는 빠른 운동 조절"**
→ 빠른 운동(피드백 불가)을 조절하는 것은 **회상도식**, 재인도식은 피드백으로 오차 확인

❌ **"준비단계 = 6개월 이내에 운동 시작 의도"**
→ 이는 **관심(숙고)단계**, 준비단계는 **1개월 이내 시작 or 현재 불규칙 운동 중**

❌ **"분산연습 = 기술을 하위단위로 나누어 연습"**
→ 이는 **분습법(부분화)**, 분산연습은 연습 사이에 충분한 **휴식이 있는 시간 구성**

❌ **"링겔만 효과의 원인 = 유능감 손실"**
→ **동기 손실(사회적 태만)**이 원인

❌ **"규정행동 = 리더에게 규정된 행동 / 선호행동 = 리더가 선호하는 행동"**
→ 규정행동 = **조직이** 리더에게 기대하는 행동, 선호행동 = **선수(구성원)가** 리더에게 바라는 행동

❌ **"심리기술훈련은 즉각적 효과를 줄 수 있다"**
→ 장기적·체계적 훈련 필요, **즉각 효과 없음**

❌ **"고원현상 = 협응 구조가 완성되어 질적 변화 없는 시기"**
→ 수행은 정체되어 보이지만, **내부적으로는 더 높은 단계로 가기 위한 질적 변화가 진행 중**

❌ **"계통발생적 행동 = 개체발생적 행동"**
→ 계통발생: 걷기·뛰기 등 **인류 보편** / 개체발생: 수영·자전거 등 **환경·연습으로 개인 발달**

❌ **"AAASP 윤리: 알고 지내는 사람과 전문적으로 진행한다"**
→ 이중관계 위험으로 **금지**. 지인과 상담은 객관성 상실 위험

## 섹션 4: 최근 3년 신출제 경향

### 2023
- **젠타일(Gentile) 2차원 운동기술분류** 신규 등장 (환경 안정성 × 동작 기능)
- **뉴웰(Newell) 제한요소** 신규 (개인·과제·환경 3분류)
- **공동작용(synergy)** — 번스타인 자유도 해결책
- **학습된 무기력 + 귀인 재훈련** 심화
- **몰입(Flow)** — 칙센트미하이 첫 등장

### 2024
- **피들러(Fiedler) 상황부합 리더십** 신규 (고/저 → 과제지향, 중 → 관계지향)
- **천장 효과·바닥 효과** 신규 (과제 난이도 측정 한계)
- **개방회로·폐쇄회로이론** 심화 (기억흔적·지각흔적 = 아담스의 폐쇄회로)
- **절차적 기억** — 기억 유형 세분화 문제

### 2025
- **모노아민 가설** 신규 (도파민·세로토닌·노르에피네프린 → 우울증 개선)
- **그리피스(Griffith)** — 스포츠심리학 역사 인물 첫 등장
- **베일리(Vealey) 스포츠자신감 원천** 신규 (성취경험·사회적 분위기·자기조절)
- **추동이론(Drive Theory)** 재등장 (각성 수준과 수행 비례)
- **맥락간섭 심화** — 구획·계열·무선 3종 직접 비교

**트렌드 요약**: 이론가 이름 + 모델 구성요소 매핑 문제가 지속 증가; 2023년부터 운동제어 심층 이론(번스타인·GMP·동적시스템) 매년 출제; 2025년부터 스포츠심리학 역사·기원 유형 등장.

## 섹션 5: 압축 암기 공식

```
[학습 3단계]
피츠&포스너 = 인지(개념이해) → 연합(오류수정) → 자동화(일관/안정)
번스타인    = 자유도 고정 → 자유도 풀림 → 반작용 활용

[발달 단계]
갈라휴 7단계 = 반사→초기→기본→스포츠기술→성장세련→최고수행→퇴보

[귀인 3×3]
능력 = 내/안정/불가   노력 = 내/불안정/가능   운 = 외/불안정/불가   난이도 = 외/안정/불가

[SDT 연속체]
무동기 → 외적 → 의무감(introjected) → 확인(identified) → 통합(integrated) → 내적

[TTM 5단계]
무관심(6mo×) → 관심(6mo내) → 준비(1mo내) → 실천 → 유지

[자기효능감 정보원 순위]
성취경험 > 대리경험 > 언어적 설득 > 신체/정서 상태

[반응시간 법칙]
Hick  = 자극-반응 대안 수↑ → 반응시간↑
Fitts = 거리↑ or 목표크기↓ → 움직임 시간↑
PRP   = 1차 처리 중 2차 자극 → 2차 반응 지연 (페이크 원리)

[GMP 매개변수]
불변 = 요소순서 + 상대적타이밍 + 상대적 힘
가변 = 전체 시간 + 전체 힘 + 선택 근육

[이완기법 4종]
바이오피드백(생체신호 시각화) / 자생훈련(무거움·따뜻함) / 점진적이완(근육수축) / 체계적둔감화(불안계층, Wolpe)

[연습구조 맥락간섭 순위]
낮음: 구획연습(blocked) → 중간: 계열연습(serial) → 높음: 무선연습(random)
※ 장기 학습·전이는 무선연습 우수, 초기 수행은 구획연습 유리

[공격성 이론 순서]
본능이론(내재 에너지) → 좌절-공격 가설(좌절→무조건) → 수정 좌절-공격(좌절+단서) → 사회학습이론(관찰·강화)

[피들러 상황부합]
고통제·저통제 → 과제지향리더 효과  /  중통제 → 관계지향리더 효과
```
