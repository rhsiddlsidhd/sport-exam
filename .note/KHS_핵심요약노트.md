## 체육사 방법론 · 🔥🔥🔥

**[비교형]**

| 구분 | 사관(史觀) | 사료(史料) |
|------|-----------|-----------|
| 정의 | 체육 역사를 바라보는 역사가의 관점·해석틀·사상 | 체육·스포츠 과거 사실을 담은 구체적 자료 |
| 종류 | 유물사관, 관념사관, 진보사관, 순환사관 | 물적·문헌·구술·구전 사료 |
| 기출 오답 | "과거 사실 자체를 기록한 것" → **사료** | - |

**[목록형] 사료 분류**
- **물적 사료**: 유물·유적·벽화 (각저총 벽화, 무용총 수렵도, 메달·경기복장 등)
- **문헌 사료**: 문서에 기록된 텍스트 (무예도보통지, 삼국사기, 회고록, 잡지 등)
- **구술 사료**: 생존자 증언·회고담
- **구전 사료**: 입으로 전해지는 민요·전설·설화 (트로피·메달은 구전 사료 ✗ → 물적 사료)

**[비교형] 나현성 시대구분**

| 기준점 | 이전 | 이후 |
|--------|------|------|
| **갑오경장(1894)** | 전통 체육 (무예 중심) | 근대 체육 (교육입국조서·학교 중심) |
| 광복(1945) | 일제강점기 | 현대 체육 |

> ⚠️ **주의** "광복을 전후로 고대체육/전통체육 구분" → 틀림. **갑오경장**이 전통↔근대 경계. "역사 기술은 가치 평가가 사실 확인보다 우선" → 틀림. 사실 확인이 선행.

## 화랑도 (신라) · 🔥🔥🔥

**[개념-설명-예시형]**

| 항목 | 내용 |
|------|------|
| 조직 체계화 | **진흥왕** 때 국가 공인 단체로 정비 |
| 핵심 윤리 | **세속오계** (원광법사 제정) — 사군이충·사친이효·교우이신·살생유택·임전무퇴 |
| 교육 방법 | 입산수행, **편력**(명산대천 야외수련), 심신일체론적 전인교육 |
| 사상적 배경 | 신체미 숭배, 국가주의, **불국토 사상** (불교적 나라 수호) |
| 구성 | 화랑(진골귀족 자제) + 낭도(평민) — 범계급적 조직, 핵심은 귀족 |

> ⚠️ **주의** 화랑도의 도의 교육 = **세속오계** (삼강오륜 ✗ → 삼강오륜은 조선 성리학). "서민층만을 대상" → 틀림. 불국토사상은 **신라 화랑도** 배경 (조선 궁술과 무관).

## 방응·격구·수박·민속놀이 계층 구분 · 🔥🔥🔥

**[비교형] 계층별 민속놀이**

| 구분 | 귀족·지배층 | 서민·대중 |
|------|-------------|---------|
| 수렵 | **방응(放鷹)** = 매사냥 (응방도감 관장, 고려) | - |
| 구기 | **격구(擊毬)** = 말 타고 채로 공치기 (기격구) | **축국(蹴鞠)** = 가죽 공 차기 (김유신·김춘추) |
| 격투 | **수박(手搏)** = 맨손 격투 (고려 무신정권 인재선발) | **씨름(각저·각력)** = 맞잡고 힘 겨루기 |
| 기타 | - | **추천(鞦韆)** = 그네뛰기 (단오, 여성) |
| 날리기 | - | **풍연(風鳶)** = 연날리기 (남녀노소 대중) |
| 돌싸움 | - | **석전(石戰)** = 돌싸움 (군사훈련 겸 민속) |
| 줄당기기 | - | **삭전·갈전(索戰·葛戰)** = 줄다리기 (마을 공동 풍농 기원) |

**[개념-설명-예시형] 수박(手搏) 핵심 정리**

| 항목 | 내용 |
|------|------|
| 시대 | 삼국시대~고려 |
| 성격 | 관람형 무예 경기, 무인 인재선발 수단 |
| 계기 | **보현원 수박희 사건(1170)** → 무신정변 촉발 |
| 관장기관 | 수박과 무관 — 응방도감은 **방응(매사냥)** 담당 |

> ⚠️ **주의** 방응 = 매사냥 = **귀족·왕실**, 풍연(연날리기) = **대중 놀이** (귀족 ✗). 격구(기격구) = **귀족** (서민 ✗). 추천 = **그네뛰기** (널뛰기 ✗). 석전 ≠ 공놀이 (축국이 공차기).

## 조선 훈련원·궁술·무과 · 🔥🔥

**[개념-설명-예시형] 훈련원(訓鍊院)**

| 항목 | 내용 |
|------|------|
| 성격 | 군사 무예 교육·훈련·무과 시험 담당 **행정·교육 기관** |
| 주요 업무 | 무예 훈련, **병서(무경칠서 등) 습독**, 군사 **시재(試才)** |
| 아닌 것 | 친위 부대 ✗ (친위대는 내금위 등), 성리학 교육 ✗ (성균관 업무) |

**[개념-설명-예시형] 조선 궁술(弓術)**

| 항목 | 내용 |
|------|------|
| 육예 분류 | 육예(禮·樂·**射**·御·書·數) 중 **사(射)** |
| 어(御) | 말타기·수레 몰기 ≠ 활쏘기 |
| 의례 형식 | **대사례(大射禮)**: 왕·신하 간 예의 있는 활쏘기 / **향사례(鄕射禮)**: 선비 간 활쏘기 |
| 정신적 배경 | **학사사상(學射思想)**: 덕성 함양 위한 활쏘기 (불국토사상 ✗ → 신라 화랑도) |
| 장소 | **사정(射亭)** — 전국 관설·민간 설치 (오운정·등룡정 등) |

**[단계-순서형] 조선 무과 3단계**

| 단계 | 명칭 | 내용 |
|------|------|------|
| 1 | **초시(初試)** | 예비 시험 |
| 2 | **복시(覆試)** | 본 시험 — 무예 실기 + **강서(병서 시험)** |
| 3 | **전시(殿試)** | 국왕 앞 최종 순위 결정 |

> ⚠️ **주의** 궁술 = 육예의 **사(射)** (어御 ✗). 무과 = 초시·복시·전시 **3단계** (예조·음양과 ✗ → 훈련원·병조 주관). 소과/대과는 **문과** 방식.

## 교육입국조서·갑오경장 · 🔥🔥🔥

**[개념-설명-예시형]**

| 항목 | 내용 |
|------|------|
| 반포 | 1895년 **고종** |
| 핵심 내용 | **삼양(三養)** 강조 — 표기 순서: **덕양(德養) → 지양(智養) → 체양(體養)** |
| 의의 | 체육을 국가 근대화 필수 요소로 공인 |
| 시대 구분 | 갑오경장(1894) 이후 = 교육입국조서(1895) → 근대 체육 시작 |

> ⚠️ **주의** 삼양 순서 = **덕·지·체** (체·덕·지 ✗). 교육입국조서 = **고종(1895)** 반포. "갑오경장 이전 = 전통 체육, 이후 = 근대 체육" 이 기준은 나현성의 시대구분.

## 개화기 학교 · 🔥🔥🔥

**[비교형] 핵심 학교 4대 비교**

| 학교 | 설립자 | 연도 | 특징 |
|------|--------|------|------|
| **원산학사(元山學舍)** | 개화파 관리+원산 주민 | 1883 | 최초 근대 사립학교; 무예반(200명) > 문예반(50명); **무비자강** |

| **대성학교** | 안창호 | 1907 | 민족 사립학교 |

> ⚠️ **주의** 이화학당 설립자 = **스크랜턴** (아펜젤러 ✗). 배재학당 설립자 = **아펜젤러** (스크랜턴 ✗). 원산학사 = **최초 근대 사립학교(1883)** (배재학당 ✗). 오산학교 = **이승훈** (안창호 = 대성학교).

> ⚠️ **주의** 최초 야구·농구 소개 = **질레트** (노백린 ✗). 전조선야구대회(1920) = **조선체육회**의 첫 사업 (YMCA ✗).

## 조선체육회 역사·성격 · 🔥🔥🔥

**[목록형] 조선체육회 핵심 사실**
- **1920년 창립** — 동아일보사 후원, 고려구락부 모태, 일본 유학 출신+국내 체육인
- 설립 목적: 민족의 체육 **지도·장려**, 일제에 대항하는 민족 체육
- 첫 사업: **제1회 전조선야구대회(1920)** → 전국체육대회의 효시
- 1920년대 활동: 육상경기위원회, 운동경기 규칙 편찬, **운동구점** 운영
- 1934년: 전조선종합경기대회 개최 (종합체전 효시)
- **1938년 강제 해산** — 조선체육협회에 통합

**[비교형] 조선체육회 vs 조선체육협회**

| 구분 | 조선체육회(1920) | 조선체육협회(1914) |
|------|-----------------|-------------------|
| 성격 | 민족 자주 체육 단체 | 일제 관변·통제 단체 |
| 설립 배경 | 동아일보 후원, 민족 운동 | 일제가 조선 체육 장악 목적 |
| 후원 | **동아일보사** | 총독부·경성일보 |

> ⚠️ **주의** 조선체육회 후원 = **동아일보** (경성일보 ✗ — 경성일보는 총독부 기관지). 조선체육협회(1914) ≠ 조선체육회(1920). **협회가 6년 먼저 설립**.
