## 섹션 2: 토픽별 핵심 개념

### 🥇 토픽 1: 운동학습 (22회)

#### 운동학습 정의와 특성
- **정의**: 연습과 경험에 의한 **비교적 영구적인** 운동 수행 능력의 내적 변화
- **직접 관찰 불가** — 수행(Performance)을 통해 간접 추론 (2019·2025 출제)
- 성숙·동기·피로에 의한 **일시적 변화는 학습 아님**
- **신경가소성(neural plasticity)** 수반

#### 운동학습 4대 지표

| 지표 | 설명 |
|------|------|
| **향상(Improvement)** | 연습에 따라 절대적 수행 점수 상승 |
| **안정성/일관성(Stability)** | 수행의 기복(변산성)이 줄어드는 것 |
| **파지(Retention)** | 연습 중단 후에도 기술 유지 |
| **전이(Transfer)** | 새 환경·유사 기술로 학습 적용 |

> **파지 검사** = 같은 환경·시간 경과 후 유지 확인 / **전이 검사** = 새로운 환경·조건에서의 적용 확인

#### 피츠(Fitts) & 포스너(Posner) 운동학습 3단계

| 단계 | 특징 |
|------|------|
| **인지 단계** | 동작의 개념·원리 이해, "어떻게 하는가" 파악 |
| **연합 단계** | 오류 탐지·수정 가능, 기술 정교화 진행 |
| **자동화 단계** | 의식적 노력 없이 안정적·일관된 수행 |

> **함정:** 순서 반드시 암기: 인지 → 연합 → 자동화. "오류를 스스로 탐지하고 수정" = 연합 (인지 아님)

#### 번스타인(Bernstein) 자유도(DOF) 3단계

| 단계 | 전략 |
|------|------|
| **초보자** | 자유도 **고정(Freezing)** — 관절 굳혀 단순화 |
| **숙련자** | 자유도 **해방(Releasing)** — 관절 풀어 유연한 동작 |
| **전문가** | 외부 물리력 **활용(Exploiting)** — 중력·반발력 이용 |

> **함정:** 초보자 = "자유도 고정" (소멸/활용 아님 — 2022 출제)

#### 연습 방법 비교

| 구분 | 특징 | 파지/전이 |
|------|------|----------|
| **집중연습** | 연습 > 휴식 | 단기 수행↑, 피로↑ |
| **분산연습** | 연습 < 휴식 | 피로 적음, 효율적 |
| **구획(블록)연습** | 한 기술 집중 반복 (서브/서브/서브) | 초기 수행↑, 파지↓ |
| **무선(랜덤)연습** | 여러 기술 무작위 혼합 (서브/토스/언더) | 초기 수행↓, **파지·전이↑** |
| **가변연습** | 한 기술 내 조건 변화 | 전이↑ |

> **맥락간섭 효과**: 높을수록(=무선연습) **파지·전이에 유리** (2020·2022·2023 출제)

#### 피드백 유형

| 구분 | 내용 | 예시 |
|------|------|------|
| **내재적 피드백** | 감각기관(시각·청각·촉각)으로 스스로 획득 | 공이 맞는 느낌 |
| **보강적 피드백** | 외부에서 추가 제공 | 지도자·장비 |
| └ **KR(결과지식)** | 수행 **결과** 정보 ("3미터 빗나갔다") | 점수, 기록 |
| └ **KP(수행지식)** | 동작 **과정** 정보 ("백스윙이 짧다") | 자세 교정 |

> **함정:** 시각·청각·촉각 = **내재적** (보강적 아님 — 2021 출제)
> **요약 피드백**: 여러 번 수행 후 한꺼번에 제공. **범위 피드백**: 오차 허용 범위 기준 제공

#### 관찰학습 (Observational Learning)
- **숙련 모델**: 완벽한 기준 제공
- **학습 모델**: 학습자와 유사 수준, 오류 수정 과정 보여줌 → 파지·전이에 효과적 (2022·2025 출제)

> **함정:** "숙련된 모델만 긍정적" → ❌ 학습 모델도 효과적

### 🥈 토픽 2: 운동제어 (19회)

#### 스포츠심리학 하위 분야

| 분야 | 연구 내용 |
|------|----------|
| **운동제어** | 동작 생성·조절 원리 (정보 처리) |
| **운동학습** | 연습으로 인한 영구적 수행 변화 |
| **운동발달** | 전 생애에 걸친 움직임의 변화 |
| **스포츠심리학** | 심리 요인이 수행에 미치는 영향 |
| **건강심리학** | 운동과 정신건강 |

#### 정보처리 3단계

`자극확인(감각·지각)` → `반응선택` → `반응실행(운동 프로그래밍)`

| 단계 | 핵심 |
|------|------|
| **자극확인** | 정보 감지·확인, 의미 부여 |
| **반응선택** | 가능한 반응 중 하나 결정 |
| **반응실행** | 근육 명령 조직화, 실제 동작 준비 |

> "실제 움직임 생성 위해 동작 조직화" = **반응실행** (2019 출제)
> 100m 스타트 반응시간 < 배구 리시브: 100m는 **자극확인 단계** 소요 시간 짧음 (2020 출제)

#### 반응시간 3종류

| 종류 | 설명 |
|------|------|
| **단순반응시간** | 1자극 → 1반응 |
| **선택반응시간** | N자극 → 각각 다른 N반응 |
| **변별반응시간** | N자극 중 특정 1자극에만 반응 |

> **함정:** "자유반응시간" → 존재하지 않는 유형 (2019 출제)

#### 시간 지표 구분

- **반응시간**: 자극 제시 → 반응 **시작** (신경 처리 시간)
- **움직임 시간**: 반응 **시작 → 종료** (근육 수행 시간)
- **전체 반응시간** = 반응시간 + 움직임 시간 (2020 출제)

#### **힉스의 법칙**: 자극-반응 대안 수 증가 → 반응시간 증가 (2024 출제)

#### 운동기술 일차원적 분류

| 기준 | 분류 |
|------|------|
| **근육 크기** | 대근운동기술 / 소근운동기술 |
| **환경 안정성** | 폐쇄운동기술 / 개방운동기술 |
| **연속성** | 불연속 / 계열 / 연속 운동기술 |

> **함정:** "과제의 난이도" → 일차원적 분류 기준 **아님** (2019 출제)
> 테니스 랠리 = 환경 변화 → **개방운동기술** (2025 출제)

#### 다이나믹 시스템 이론 (동역학체계)
- **상변이**: 협응 형태가 질적으로 바뀌는 현상 = **비선형** 원리 (선형 아님 — 2020 출제)
- **제어변수**: 상변이를 유발하는 외부 요인 (속도, 무게)
- **질서변수**: 협응의 형태를 나타내는 변수 (상대적 위상)
- **자기조직**: 중앙 명령 없이 구성 요소들이 스스로 질서 형성

### 🥉 토픽 3: 동기 이론 (16회)

#### 데시(Deci) & 라이언(Ryan) 자기결정이론(SDT) — 핵심 중 핵심

| 동기 유형 | 자율성 | 설명 | 예시 |
|----------|--------|------|------|
| **무동기** | 없음 | 아무 의욕 없음 | — |
| **외적규제** | 낮음 | 상금·처벌 회피 | 상을 받으려고 |
| **의무감규제(내사규제)** | ↑ | 죄책감·타인시선 압박 | 평가받기 싫어서 |
| **확인규제** | ↑↑ | 활동의 중요성 스스로 인정 | 건강을 위해 |
| **통합규제** | ↑↑↑ | 자아 정체성과 통합 | — |
| **내적규제** | 최고 | 활동 자체의 즐거움 | 재미있어서 |

> **함정:** "평가받기 싫어서" = **의무감규제** (외적규제 아님 — 2019·2021 출제)
> "건강을 위해 스스로 조깅" = **확인규제** (2021·2025 출제)
> 3대 심리 욕구: **자율성·유능감·관계성** — 외적 보상 강조는 자율성 훼손 → 내적동기 저해

#### 인지평가이론 (CET — SDT의 하위 이론)
- 3욕구 중 자율성·유능감에 초점
- 외부 보상이 **통제적**으로 느껴지면 내적 동기 감소 ("과잉 정당화 효과")
- 보상 강조 → 내적동기 저하 (2021 출제)

#### 와이너(Weiner) 귀인 이론 — 3차원

| 귀인 요소 | 내적/외적 | 안정/불안정 | 통제가능/불가 |
|----------|-----------|------------|--------------|
| **능력** | 내적 | 안정 | 불가 |
| **노력** | 내적 | **불안정** | **가능** |
| **과제 난이도** | 외적 | 안정 | 불가 |
| **운** | 외적 | 불안정 | 불가 |
| **상대 선수 능력** | 외적 | 안정 | 불가 |

> **귀인 재훈련**: 실패 원인을 "**내적·불안정·통제가능**한 노력"으로 귀인하도록 유도
> 실패를 "능력(안정·내적·불가)"으로 귀인하면 무기력 유발 → 운동 포기

#### 성취목표성향 이론

| 성향 | 기준 | 특징 |
|------|------|------|
| **과제목표성향(숙달)** | 자기 향상 | 노력 귀인, 실패에도 도전, 적절 난이도 선택 |
| **자기목표성향(수행)** | 타인 비교 | 승패 중시, 실패 시 자존감 방어(능력·운 귀인) |

> **함정:** 과제목표성향 → "패배 이유 = 자신의 **노력 부족**" (상대 실수 아님 — 2019 출제)

### 🎯 토픽 4: 심리기술훈련 (15회)

#### 목표설정 3유형

| 유형 | 기준 | 예시 |
|------|------|------|
| **결과 목표** | 승패·순위 (타인 비교) | "대회에서 우승한다" |
| **수행 목표** | 자신의 기록·수치 | "슈팅 성공률 80%", "100m 12초" |
| **과정 목표** | 동작·기술의 절차 | "백스윙 시 팔꿈치 고정", "공 끝까지 보기" |

> **함정:** "우승" = **결과목표** (수행목표 아님 — 2020 출제)
> 결과목표는 통제 불가; 수행·과정 목표는 통제 가능 → 권장

#### SMART 원칙
구체적(Specific) · 측정가능(Measurable) · 행동지향적(Action-oriented) · 현실적(Realistic) · 시간제한(Time-bound)

#### 심상 훈련 (Imagery)

| 요소 | 설명 |
|------|------|
| **선명도(Vividness)** | 오감을 동원해 실제처럼 생생하게 떠올리는 능력 |
| **조절력(Controllability)** | 이미지를 의도대로 변형·통제하는 능력 |

| 관점 | 설명 |
|------|------|
| **내적 심상** | 1인칭 시점 — 내가 실제로 수행하는 느낌 |
| **외적 심상** | 3인칭 시점 — 제3자가 나를 보는 장면 |

> "부정적 수행 → 성공 이미지로 바꿈" = **조절력** (2020 출제)
> "실제처럼 자신감·흥분 시각화" = **선명도** (2020 출제)

#### 루틴 (Pre-performance Routine)
- 최상 수행 위한 **습관화된 절차** (심상·혼잣말 포함 가능)
- 불안 감소, 집중력 증대, 낯선 환경에서도 안정감 제공
- ❌ **시합 당일 수정** → 루틴의 본질을 파괴 (2019 출제)
- 루틴 수행 중에는 **의식적 처리 감소** (자동화 촉진)

#### 자가진단(혼잣말, Self-talk)
- 주의 집중, 자신감↑, 동기 유발
- ❌ "부정적 혼잣말은 언제나 배제" → **아님** — 상황에 따라 자극 효과 가능 (2022 출제)
- 기술 습득 초기: 인지적 단서 제공 혼잣말 효과적

#### 각성 조절 기법

| 기법 | 설명 |
|------|------|
| **바이오피드백** | 생체 신호(심박수·근전도)를 시각/청각 정보로 변환해 스스로 조절 학습 |
| **점진적 이완** | 근육의 긴장-이완 반복으로 신체 긴장 해소 |
| **자생훈련** | 신체 부위의 온감·중량감 유도 (자기 최면적) |
| **인지 재구성** | 부정적 생각·상황을 긍정적으로 재해석 |
| **사고 정지** | 부정적 생각 발생 시 즉각 차단 |

### 🏃 토픽 5: 운동 행동·건강심리 (14회)

#### 프로차스카(Prochaska) 범이론적 모형 (변화단계이론) — 매년 출제

| 단계 | 핵심 내용 | 기간 |
|------|----------|------|
| **무관심(전숙고)** | 운동 참여 의도 없음, 6개월 내 시작 계획 없음 | — |
| **관심(숙고)** | 6개월 **이내** 운동 시작 의도 | — |
| **준비** | 1개월 **이내** 시작 계획, 비정기적 참여 | — |
| **실천** | 규칙적 운동 중, 6개월 **미만** | < 6개월 |
| **유지** | 규칙적 운동 지속 | 6개월 **이상** |

> **함정:** "준비 단계 = 6개월 이내 시작 의도" → ❌ 그건 **관심 단계** (2020·2022 출제)
> "준비 단계 = **1개월** 이내" 암기 (2024·2025 출제)
> **자기효능감**: 유지 단계로 갈수록 높아짐

- **의사결정 균형**: 운동 혜택(장점) vs 장애 요인(단점) 비교 평가
- **변화 과정**: 인지과정(의식고양·자기재평가 등) + 행동과정(자극 통제·강화 관리 등)
- 무관심→관심 전략: **의식 고양** (정보 제공) (2023 출제)

#### 아이젠(Ajzen) 계획된 행동이론 (TPB)
`태도` + `주관적 규범` + `행동통제인식` → **의도** → 행동

> **함정:** "동기, 자신감" = 구성 요소 아님 (2021 출제)

#### 행동수정 전략 (Behavior Modification)

| 전략 | 정의 | 예시 |
|------|------|------|
| **정적 강화** | 좋은 자극 **부여** → 행동↑ | 칭찬 스티커 |
| **부적 강화** | 싫은 자극 **제거** → 행동↑ | 숙제 면제 |
| **정적 처벌** | 싫은 자극 **부여** → 행동↓ | 꾸중, 벌점 |
| **부적 처벌** | 좋은 자극 **제거** → 행동↓ | 출전 시간 박탈 |

> "폭력 시 **출전 시간 제한**" = **부적 처벌** (좋은 것 빼앗음 — 2019 출제)

#### 운동 애착(Exercise Adherence) 증진 전략
자율적 선택권 부여 / 개별 피드백 / 사회적 지지 / 환경 단서 활용

> **함정:** "지도자가 운동을 선택해줌" → ❌ **자율성 저해** (2019 출제)

#### 운동중독 (Exercise Addiction)
- **긍정적 중독**: 건강 유지, 자기통제, 삶에 활력
- **부정적 중독**: 강박적 참여, 부상에도 지속, 일상·관계 파괴
- **금단 증상**: 짜증·불안·수행 저하 — 긍정적 효과는 금단증상 아님 (2023 출제)

### 📉 토픽 6: 불안과 각성 이론 (11회)

#### 각성-수행 관계 4대 이론

| 이론 | 관계 | 핵심 |
|------|------|------|
| **추동 이론(Hull-Spence)** | **직선** (정비례) | 각성↑ → 수행↑. 숙련자에게 유리(우세반응 강화) |
| **역U자 이론** | 역U자 곡선 | 중간 각성에서 최적 수행. 과제 복잡도에 따라 최적점 다름 |
| **반전 이론(Kerr)** | 각성 해석에 따라 다름 | 동일 각성도 "흥분(긍정)" or "불안(부정)"으로 **반전** 가능. 인지적 해석이 결정 |
| **카타스트로피 이론(Hardy)** | 인지불안 높을 때 급락 | 인지불안 高 + 생리적 각성 임계점 초과 → 수행이 **급격히 붕괴** |
| **IZOF(한닌)** | 개인마다 다름 | 각 선수마다 고유한 **최적수행 불안 구역** 존재 |

> **함정:** "각성↑ → 항상 수행↑" = **추동이론** (역U자 아님)
> "각성 수준 해석에 따라 다름" = **반전이론** (2019 출제)
> "인지 불안 높을 때 급격한 붕괴" = **카타스트로피** (2023 출제)

#### 불안 유형

| 구분 | 특징 |
|------|------|
| **특성불안** | 성격적·안정적, 어떤 상황에서도 지속, SCAT으로 측정 |
| **상태불안** | 특정 상황에 반응해 **일시적**으로 나타남, 상황 변화 시 소멸 |

- 상태불안 2차원: **인지불안**(걱정·우려) + **신체불안**(생리적 긴장)
- **지각 협소화(Perceptual Narrowing)**: 고각성 → 주의 폭 극도로 좁아짐 (터널 시야) (2020 출제)

#### 다차원불안이론 (Martens)
- 인지불안 → 수행과 **부적(-)** 선형 관계
- 신체불안 → 수행과 **역U자** 관계

### 👥 토픽 7: 집단 역동 (10회)

#### 캐론(Carron) 팀 응집력 모형

**4대 선행 요인**: 환경 요인 / 개인 요인 / 리더십 요인 / 팀 요인

> **함정:** "발달 요인" = 존재하지 않는 요인 (2021 출제)
> "팀의 규모" = **환경 요인** (팀 요인 아님 — 2019 출제)
> 팀 요인에 해당: 팀의 목표, 팀의 능력(규범·정체성 포함)

**응집력 2차원**: 과제 응집력(목표 달성) + 사회 응집력(대인 관계)
**응집력과 경기력**: 상호 **순환적 관계** (2025 출제)

> 상호작용 종목(농구·축구): 과제 응집력과 수행의 관계 더 높음
> 공존 종목(육상·수영): 영향 적음

#### 사회적 태만 (Social Loafing) / 링겔만 효과

| 개념 | 설명 |
|------|------|
| **링겔만 효과** | 집단 인원 증가 → 1인당 수행 감소 현상 |
| **원인** | **동기 손실** (개별 노력이 가려짐) |
| **극복 전략** | 개별 수행 기록, 책임감 부여, 팀 규모 축소 |

> **함정:** "유능감 손실"은 원인 분류 상 표준 용어 아님; 원인 = **동기 손실** (2020 출제)
> "팀 규모 축소" = 태만 극복 전략 (2023·2024 출제)

### 💬 토픽 8: 상담 윤리·기법 (7회)

### 🏅 토픽 10: 운동발달 (6회)

#### 발달 단계 (Gallahue 기반)

`반사 움직임` → `초기 움직임` → `기본 움직임` → `스포츠 기술` → `성장세련·최고수행` → **퇴보**

- **기본 움직임 단계**: 2~7세, 달리기·던지기·뛰어넘기 등 기본 운동 패턴 형성 (2023·2024 출제)
- **퇴보 단계**: 노화로 수행 능력 하강 (2025 출제)

#### 발달 원리

| 원리 | 방향 |
|------|------|
| **두미 발달** | 머리에서 발 방향 (두부→족부) |
| **중말 발달(근원)** | 중심에서 말단 방향 (몸통→손끝) |
| **계통발생적 발달** | 종 전체의 진화 순서 |
| **개체발생적 발달** | 개인 생애에 걸친 발달 |

### 👔 토픽 11: 리더십 (5회)

#### 첼라두라이(Chelladurai) 다차원 리더십 모형
- **3가지 리더십 행동**: 규정된 행동 / 선호되는 행동 / **실제 행동**
- 3가지가 **일치**할 때 선수 만족·경기력 최대화 (2024·2025 출제)
- 리더십 행동 영향 요인: 상황적 특성 / 지도자 특성 / 구성원 특성

#### 바람직하지 않은 리더십 행동 (2020 출제)
- ❌ 과도한 자신감 부여 → 자만심·실패 시 큰 좌절
- ✅ 개별 관심·시간 할애, 목표 수립 지원, 주의산만 요인 파악·지도

### 🧠 토픽 12: 성격 구조 (4회)

#### 홀랜더(Hollander) 성격 3층 구조

| 층위 | 특징 | 변화 가능성 |
|------|------|------------|
| **심리적 핵** | 가장 깊은 핵심 가치·정체성 | **가장 안정적**, 환경 영향 최소 |
| **전형적 반응** | 상황에 대한 통상적 반응 패턴 | 중간 |
| **역할 행동** | 상황에 따라 유연하게 변하는 행동 | **가장 유동적** |

> **함정:** "심리적 핵 = 환경 영향 가장 많이 받음" → ❌ 가장 **안정적** (2022·2024 출제)

#### 빙산형 프로파일 (Iceberg Profile)
- 우수 선수: **활력↑**, 긴장·우울·분노·피로·혼돈 등 부정 정서↓
- 비교집단보다 긍정 정서가 두드러지게 높은 형태

## 섹션 3: 함정 선지 총정리

| ❌ 함정 선지 | ✅ 올바른 내용 |
|------------|-------------|
| "운동학습 과정을 직접 관찰할 수 있다" | 직접 **관찰 불가**, 수행 결과로 추론 |
| "상변이는 선형 원리를 따른다" | **비선형** 원리 |
| "초보자는 자유도를 해방한다" | 초보자는 자유도 **고정** |
| "분산연습 = 연습 > 휴식" | 분산연습은 **연습 < 휴식** |
| "구획연습이 파지·전이에 유리" | **무선연습**이 파지·전이 유리 |
| "숙련 모델만 긍정적" | **학습 모델**도 파지·전이에 효과적 |
| "과제의 난이도 = 일차원 분류 기준" | 난이도는 **분류 기준 아님** |
| "자유반응시간이 있다" | **존재하지 않는** 유형 |
| "시각·청각·촉각 = 보강적 피드백" | 감각 기관 정보 = **내재적 피드백** |
| "평가받기 싫어서 = 확인규제" | **의무감규제**  |
| "건강 위해 스스로 참여 = 외적규제" | **확인규제** |
| "외적 보상 강조 → 내적동기 증가" | 외적 보상 강조 → 자율성 훼손 → **내적동기 감소** |
| "준비단계 = 6개월 이내 시작 의도" | 6개월 이내는 **관심단계**, 준비는 **1개월 이내** |
| "우승 목표 = 수행목표" | 우승 = **결과목표** |
| "루틴을 시합 당일 수정한다" | 루틴은 **시합 당일 수정 금지** |
| "부정적 혼잣말은 언제나 배제" | 상황에 따라 **자극 효과** 가능 |
| "심리적 핵 = 환경 영향 가장 많이 받음" | 심리적 핵은 **가장 안정적** |
| "팀의 규모 = 팀 요인" | 팀의 규모 = **환경 요인** |
| "발달 요인 = 응집력 선행 요인" | **발달 요인은 없음** (환경·개인·리더십·팀) |
| "폭력 시 출전 시간 박탈 = 정적 처벌" | 좋은 것 제거 = **부적 처벌** |
| "목표설정 = 행동수정 전략" | 목표설정 = **인지적 접근** (행동수정 아님) |

## 섹션 4: 최근 3년 신출제 경향 (2023–2025)

#### 2023 신출제
- **피츠 법칙** (Fitts' Law): 목표 정밀도↑ → 이동 시간 증가
- **요약 피드백·범위 피드백** 유형 세분화
- **반두라 관찰학습 4단계** (주의→파지→행동재생→동기)
- 카타스트로피 이론 직접 출제

#### 2024 신출제
- **힉스의 법칙** (Hick's Law): 자극-반응 대안 수 ↑ → 반응시간 ↑
- **첼라두라이 다차원 리더십 모형** 구체 출제
- SMART 목표 원칙
- **부정적 운동중독** 유형 구분

#### 2025 신출제
- **가변연습**과 전이 능력 세트 출제
- **학습 무기력** 극복 (작은 성공 경험 제공)
- 응집력-경기력 **순환 관계** 강조
- 첼라두라이 **3대 행동 일치** 조건 강조

**트렌드 요약:** 기존 이론의 세분화(피드백 유형, 연습 방법) + 새 학자 추가(첼라두라이, 번스타인) + 개인-집단의 상호 관계 강조.
