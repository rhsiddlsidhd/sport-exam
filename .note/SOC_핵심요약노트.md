## 섹션 2: 토픽별 핵심 개념

### 🥇 토픽 1: 스포츠 사회화 (15회)

#### 사회화 4단계 — 매년 출제, ITEMIZED 유형 필수

| 단계 | 설명 | 예시 |
|------|------|------|
| **스포츠로의 사회화** | 처음 스포츠에 입문 | 아빠 따라 골프장 가서 배우게 됨, TV보고 축구클럽 가입 |
| **스포츠를 통한 사회화** | 참여 중 가치·규범 내면화 | 골프 배우며 매너·체력·준법정신 함양 |
| **스포츠 탈사회화** | 참여 중단 | 손목 수술 후유증으로 그만둠, 부상으로 그만둠 |
| **스포츠로의 재사회화** | 중단 후 복귀 | 은퇴 후 아카데미 원장으로 활동, 친구 권유로 테니스 시작 |

> **함정:** ㉠=탈, ㉡=통한, ㉢=로의, ㉣=재사회화 조합이 매년 반복. 반드시 ITEMIZED 지문 꼼꼼히 읽기

#### 사회학습이론 3대 요소 (레오나르드 기준)

| 요소 | 설명 |
|------|------|
| **강화(Reinforcement)** | 상과 벌을 통한 행동 변화 |
| **코칭(Coaching)** | 지도자의 직접 교수 → 지식·기술 습득, 동기 부여 |
| **관찰학습/모방** | 타인 행동 보고 모방 |

> **함정:** "스스로 상황을 판단해 변화" = **상징적 상호작용론** (사회학습이론 아님 — 2023·2025 출제)

#### 스나이더(Snyder) 사회화 전이 조건

| 조건 | O/X |
|------|-----|
| 참가의 정도 | ✅ |
| 참가의 자발성 여부 | ✅ |
| 사회화 주관자의 위신과 위력 | ✅ |
| **참가의 가치** | ❌ (전이 조건 아님 — 2023 출제) |

#### 사회화 주관자

`가족` → `학교` → `동료집단` → `지역사회` → `대중매체`

> 부모님 권유 = **가족**, 주민센터 프로그램 = **지역사회** (2025 출제)

#### 준거집단이론 함정
- 준거집단이론: 집단의 **규범과 규칙을 내면화하여 따를 때** 사회화 발생
- "집단 규칙 따르지 않아도 사회화" → ❌ (2025 오답)

### 🥈 토픽 2: 스포츠와 정치 (13회)

#### 에티즌(Eitzen) & 세이지(Sage) — 스포츠의 정치적 속성

| 속성 | 설명 | 예시 |
|------|------|------|
| **보수성** | 기존 사회 질서 유지 | — |
| **대표성** | 집단/국가를 상징 | — |
| **권력투쟁** | 자원과 권한 확보 위한 갈등 | 팀·리그·선수단체 간 권한 다툼 |
| **상호의존성** | 스포츠와 정치의 상호 필요 관계 | 선수=국위선양 ↔ 국가=혜택 제공 |

> **함정:** 아닌 것 = **상호배타성** (2022 출제). 있는 속성 4개 암기 필수

#### 정치의 스포츠 이용 3방법

| 방법 | 설명 | 예시 |
|------|------|------|
| **상징(Symbol)** | 시각적·청각적 의례로 가치 부여 | 국가 연주, 국기 의례, 선수 복장 |
| **동일화(Identification)** | 선수/팀과 감정적 일체화 | 금메달 장면에 눈물, 태권도=우리나라 |
| **조작(Manipulation)** | 정치적 관심 분산용 스포츠 활용 | 3S 정책, 프로스포츠 장려로 국민 시선 분산 |

> **함정:** "3S 정책" = 조작 (상징 아님). "금메달 내가 딴 것처럼 눈물" = 동일화 (상징화 아님)

#### 스트랭크(Strenk) 국제정치에서 스포츠 기능

| 사례 | 기능 |
|------|------|
| 1936 베를린 올림픽 | **정치이념 선전** (나치즘 우월성 과시) |
| 1971 핑퐁 외교 (미국-중국) | **외교적 도구** |
| 1972 뮌헨 올림픽 검은구월단 | **갈등 및 적대감 표출** |
| 1980 모스크바 → 서방국 보이콧 | **외교적 항의** |
| 남아공 아파르트헤이트 국제 제재 | **외교적 항의** |
| 2002 한일 월드컵 4강 | **국위 선양** |

> **함정:** 베를린 = 이념선전 ≠ 국위선양. 1980 보이콧 = 서방국; 1984 보이콧 = 동구권

#### 홀리한(Houlihan) 정부의 스포츠 개입 목적
건강 증진 / 평등 실현(Title IX) / 공공질서 보호 → **모두 해당** (2024 출제)

#### 스포츠 육성 정책 모형

| 모형 | 방향 | 설명 |
|------|------|------|
| **피라미드 모형** | 저변 → 엘리트 | 대중 참여 저변이 넓을수록 우수 선수 배출 |
| **낙수효과 모형** | 엘리트 → 저변 | 엘리트 성공이 대중 참여 유도 |
| **선순환 모형** | 상호 강화 | 엘리트↔저변이 서로 자극하며 함께 발전 |

### 🥉 토픽 3: 스포츠 일탈 (12회)

#### 머튼(Merton) 아노미 적응 유형 — 5가지 완전 암기

| 유형 | 목표 | 수단 | 예시 |
|------|------|------|------|
| **동조주의** | O | O | 규칙 내 파울, 전략적 시간 끌기 |
| **혁신주의** | O | X(불법) | 불법 약물, 승부 조작, 불법 스카우트 |
| **의례주의** | X | O | 참가에만 의의, 결과보다 내용 중시 |
| **도피주의** | X | X | 승리지상주의 염증으로 스포츠 포기 |
| **반역주의** | 대체 | 대체 | 새 목표+수단으로 체제 전복 시도 |

> **함정:** "결과보다 과정" = **의례주의** (동조주의 아님). 약물복용 = **혁신주의** (2025 출제)

#### 코클리(Coakley) 과잉동조 — 4대 윤리 규범

| 규범 | 설명 |
|------|------|
| **구분짓기 규범** | 다른 선수와 구별되기 위해 탁월성 추구 |
| **인내 규범** | 위험·고통 속에서도 경기 참여 (부상 투혼) |
| **몰입 규범** | 경기에 헌신, 최우선 순위 |
| **도전 규범** | 장애 극복, 역경 헤쳐 나감 |

- **4개 모두 옳음** (2022 출제, 정답=4번)
- 부상 투혼 = **과잉동조** + **인내 규범** (2024 황선홍·손흥민 사례)
- 과소동조 = 규범 무시/거부

#### 일탈 이론 비교

| 이론 | 학자 | 핵심 |
|------|------|------|
| **아노미 이론** | 머튼 | 목표-수단 불일치 |
| **차별교제 이론** | 서덜랜드(Sutherland) | 일탈적 타인과 교류하며 일탈 **학습** — 상호작용론적 |
| **낙인 이론** | 베커(Becker) | 타인의 부정적 반응이 자아 정체성 변화 → 이차적 일탈 |
| **집합행동 이론** | 다수 | 전염(파도처럼 확산) / 수렴(공통성향) / 규범생성(임시규칙) / 부가가치(단계적 유발) |

> **함정:** 차별교제이론 = 상호작용론 기반 (아노미이론 아님). 낙인 = 타인 반응 → 정체성 변화

#### 일탈의 순기능 vs 역기능
- 순기능: 규범 재확인, 감정 배출(아이스하키 주먹다짐), 기술 혁신(배면뛰기)
- ❌ "약물복용이 공정성 훼손" → **역기능** (순기능 아님 — 2023 출제)

#### 스미스(Smith) 경기장 폭력 4유형
`격렬한 신체 접촉` → `경계 폭력`(묵인된 규칙위반) → `유사 범죄 폭력` → `범죄 폭력`

### 📺 토픽 4: 스포츠와 미디어 (11회)

#### 버렐(Birrell) & 로이(Loy) — 미디어스포츠 수용 욕구 4유형

| 욕구 | 설명 | 예시 |
|------|------|------|
| **인지적** | 지식·정보·통계 습득 | 팀 정보 검색, 리그 순위 확인 |
| **정의적** | 흥분·감동 등 감정적 만족 | 승부의 스릴 |
| **통합적** | 공동체 의식, 유대감 | 다른 집단과 경험 공유 |
| **도피적** | 스트레스·일상 탈피 | — |

> **함정:** 2023 Q7 = 인지(㉠)·**정의**(㉡)·통합(㉢). 2025 정보검색 = 인지적. "심동적·소비적" = 없는 욕구

#### 맥퍼슨(McPherson) 미디어 이론 (사회관계이론)
- **사회관계이론**: 중요타자(가족·친구)의 스포츠 소비 행동이 개인 미디어 소비에 영향
- 비교: 개인차이론(개인 심리·욕구), 사회범주이론(소속 집단), 문화규범이론(매체가 문화 규범 형성)

#### 맥루한(McLuhan) 핫/쿨 미디어

| 구분 | 정보량 | 참여도 | 특징 |
|------|--------|--------|------|
| **핫(Hot)** | 많음(고해상도) | 낮음(수동적) | 참여 필요 없이 정보 전달 완결 |
| **쿨(Cool)** | 적음(저해상도) | 높음(능동적) | 수용자가 빈 부분 채워 넣어야 함 |

> **함정:** "핫 미디어 = 경기 진행 속도 빠름" → ❌ 빠른 경기(쿨 미디어 특성) 아님 (2022 출제)

#### 미디어와 스포츠 상호 영향 구별

| 미디어 → 스포츠 | 스포츠 → 미디어 |
|----------------|----------------|
| 규칙·일정 변경 | 방송 기술 발전(슬로우모션, 클로즈업) |
| 탁구공 색 변경(흰→주황) | 컬러TV·스마트폰 보급 촉진 |
| 경기 인구 증가 | — |

> **함정:** 탁구공 색 변경 = 미디어가 스포츠에 미친 영향. 방송기술 발전 = 스포츠가 미디어에 미친 영향 (2025 Q7)

#### 보편적 접근권 vs 독점 중계권
- **보편적 접근권**: 국민이 주요 경기를 무료/저렴하게 볼 권리
- **옐로 저널리즘**: 선수 사생활 중심, 자극적·선정적 보도

### 📊 토픽 5: 사회계층과 이동 (10회)

#### 투민(Tumin) 계층의 5대 특성

| 특성 | 설명 | 시험 사례 |
|------|------|----------|
| **역사성** | 시간에 따라 변천 | 운동선수 지위가 과거보다 향상 |
| **보편성(편재성)** | 모든 사회에 존재 | 인기·비인기 종목 구분, 체급별 대전료 차등 |
| **고착성** | 쉽게 변하지 않음 | — |
| **영향성** | 삶의 기회에 영향 | 계층에 따라 참여 빈도·종목이 다름 |
| **다양성** | 사회마다 형성 기준 다름 | — |

> **함정:** 보편성 ≠ 역사성. 인기·비인기 구분 = 보편성. 시간 변화 = 역사성 (2024 Q6 함정)

#### 투민 계층 형성 4단계
`지위의 분화` → `지위의 서열화` → `평가` → `보수부여`

#### 사회계층 이동 준거 (기든스·Giddens)

| 준거 | 유형 |
|------|------|
| **이동 방향** | 수직 이동(지위 고저 변화) / 수평 이동(지위 유지, 소속 변화) |
| **이동 주체** | 개인 이동 / 집단 이동 |
| **이동 시간** | 세대 내 이동(한 사람 생애) / 세대 간 이동(부모↔자녀) |

> **함정:** "방향에 따라 세대내/세대간" → ❌ 방향=수직/수평, 시간=세대내/세대간 (2024 Q7)
> **함정:** 부모와 자녀 지위 비교 = **세대 간** 이동 (세대내 아님 — 2025 Q12)

#### 부르디외(Bourdieu) 문화자본 3유형

| 유형 | 특징 | 스포츠 예시 |
|------|------|------------|
| **체화된 문화자본** | 몸에 밴 성향(아비투스), 양도 불가 | 테니스 기술+매너, 오랜 훈련으로 체득 |
| **객체화된 문화자본** | 물적 소유물 | 스포츠 장비 |
| **제도화된 문화자본** | 자격증·학위 | 지도자 자격증 |

#### 베블런(Veblen) 유한계급론
- 자신의 부와 지위 과시를 위한 **과시적 소비**
- 골프·요트 = 계층 표식 행위
- 투민 **영향성** + 베블런 **유한계급론** 세트 출제 (2025 Q6)

### 🌍 토픽 6: 스포츠 세계화 (10회)

#### 세계화 동인

| 동인 | 포함 여부 |
|------|----------|
| 민족주의, 제국주의 | ✅ |
| 종교 전파, 과학기술 발전 | ✅ |
| **인종차별 심화** | ❌ (세계화 저해 요인 — 2024 출제) |

#### 스포츠 세계화 결과

| 현상 | O/X |
|------|-----|
| 국제 스포츠 기구 성장(IOC, FIFA) | ✅ |
| 다국적 기업 스폰서십 증가 | ✅ |
| 전통 스포츠 세계적 확대 | ❌ → **전통 스포츠 소외** (2022 Q20) |
| 외국인 선수 증가로 국가 정체성 강화 | ❌ → **국가 정체성 약화** (2024 Q14) |

#### 세방화 (Glocalization) — 로버트슨(Robertson) 제시
- 글로벌 브랜드 + 지역 문화 결합
- 박찬호(LA다저스)로 MLB 한국 인기 상승 = 세방화
- 마오리 하카댄스를 글로벌 광고에 = 세방화 (2025 Q10)

#### 메기(Magee) & 서덴(Sugden) — 스포츠 노동이주 유형

| 유형 | 특징 |
|------|------|
| **용병형** | 오직 금전적 이익 추구, 정서적 유대 없음 |
| **개척자형** | 선교사적, 종목 전파·새 문화 경험 추구 (돈 아님) |
| **정착민형** | 영구 정착지 탐색 |
| **귀향민형** | 해외 경험 후 자국 복귀 |
| **유목민형** | 개인 취향대로 흥미로운 곳 이동 |

> **함정:** 개척자형 = **돈 아닌 종목 전파**. 돈 추구 = 용병형 (2023 Q15)

### 📱 토픽 7: 사회이론 (거시·미시) (9회)

#### 3대 이론 비교

| 이론 | 관점 | 핵심 | 스포츠 기능 |
|------|------|------|------------|
| **구조기능주의** | 거시, 안정 | 사회 시스템 유지 | 적응·목표달성·통합·체제유지 |
| **갈등이론** | 거시, 불평등 | 지배-피지배 권력 관계 | 사회갈등 유발 기능 |
| **상징적 상호작용론** | **미시**, 의미 | 개인의 능동적 의미 부여 | 하위문화·역할 형성 |

#### 파슨즈(Parsons) AGIL 모형 — 구조기능주의

| 기능 | 설명 | 스포츠 |
|------|------|--------|
| **A** - 적응(Adaptation) | 외부 환경 자원 확보·분배 | 건강 증진, 사회 소양 학습 |
| **G** - 목표성취(Goal Attainment) | 공동 목표 달성 | — |
| **I** - 통합(Integration) | 구성원 결속, 갈등 조정 | 사회통합 |
| **L** - 잠재적 체제유지(Latency) | 핵심 가치·규범 보존 | 체제유지 및 긴장 처리 |

> **함정:** AGIL = **구조기능주의** (상징적 상호작용론 아님 — 2024 Q11)

#### 스티븐슨(Stevenson) & 닉슨(Nixon) — 구조기능주의 스포츠 기능
사회·정서적 기능(ㄱ) + 사회통합(ㄷ) + 사회계층 이동(ㄹ) → 옳음
**사회갈등 유발(ㄴ) → ❌** (갈등이론 관점 — 2024 Q3)

#### 거트만(Guttmann) 근대스포츠 7대 특징
`세속화` · `평등성` · `전문화` · `합리화` · `관료화` · `수량화` · `기록의 추구`

| 특징 | 예시 |
|------|------|
| 세속화 | 신 숭배→이익·오락·보상 추구 |
| 평등성 | 인종·성별 무관 동등 참여 기회 |
| 전문화 | 포지션 세분화(선발·중간·마무리) |
| 관료화 | 국제스포츠조직이 규칙 제정·관리 |

> 자주 출제: 관료화 vs 합리화 구별. 관료화=공식 행정 체계, 합리화=효율적 규칙 정비

### 💰 토픽 8: 상업화·프로스포츠 (9회)

#### 코클리(Coakley) 상업주의에 따른 스포츠 변화

| 변화 영역 | 내용 |
|----------|------|
| **조직 변화** | 경품 추첨, 연예인 시구 등 이벤트 마케팅 |
| **목적 변화** | 아마추어리즘 → 프로페셔널리즘·흥행 추구 |
| **내용 변화** | 전후반제 → 쿼터제, 3점슛 도입 |

> **함정:** "심미적 가치 → 영웅적 가치"는 구조 변화가 아닌 오리엔테이션 변화 (2023 Q2 함정 선지)

#### 코클리 상업주의 규칙 변화 조건

| 조건 | 포함 여부 |
|------|----------|
| 경기 속도감 향상 | ✅ |
| 관중 흥미 극대화 | ✅ |
| 상업 광고 시간 확보 | ✅ |
| **득점 방법 단일화** | ❌ → 오히려 **득점 방법 다양화·득점 증가** (2024 Q10) |

#### 프로스포츠 주요 제도

| 제도 | 설명 |
|------|------|
| **드래프트(Draft)** | 신인 선수 쟁탈 방지, 계약금 인상 경쟁 억제 |
| **FA(Free Agent)** | 계약 기간 후 자유롭게 구단 선택 |
| **샐러리 캡(Salary Cap)** | 팀 전체 연봉 총액 한도 |
| **보류 조항** | 구단이 선수 권리 독점, 이적 방지 |
| **웨이버(Waiver)** | 방출 전 타 구단에 양도 의사 공시 (2023 Q9) |

#### 상업주의 스포츠 출현 3대 조건 (코클리)
자본주의 시장경제 + 소비가 장려되는 문화 + 인구 밀도 높은 대도시

> **함정:** **스태그플레이션** = 상업화 저해 요인 (조건 아님 — 2025 Q17)

## 섹션 3: 함정 선지 총정리

#### 정치·국제사회

| ❌ 함정 선지 | ✅ 올바른 내용 |
|------------|-------------|
| "3S 정책 = 상징의 방법" | 3S 정책은 **조작** |
| "금메달 내가 딴 것처럼 눈물 = 상징화" | **동일화** |
| "1984 LA올림픽 → 자유 진영 불참" | 1984 LA = **동구권 보이콧** (서방국 불참은 1980 모스크바) |
| "1936 베를린 = 검은구월단 사건" | 검은구월단 = **1972 뮌헨** |
| "스포츠의 정치적 속성 = 상호배타성" | **상호의존성** (배타성은 없는 속성) |
| "2002 월드컵 4강 = 외교적 도구" | **국위선양** |
| "1995 남아공 럭비 = 보이콧 발생" | **인종화합의 대회** (보이콧 없음) |

#### 사회화

| ❌ 함정 선지 | ✅ 올바른 내용 |
|------------|-------------|
| "스스로 상황 학습·변화 = 사회학습이론" | **상징적 상호작용론** |
| "참가의 가치 = 스나이더 전이 조건" | 전이 조건 **아님** |
| "부모와 비교 지위 변화 = 세대 내 이동" | **세대 간 이동** |
| "준거집단 규칙 안 따라도 사회화 가능" | **따를 때** 사회화 |
| "부상·방출 = 탈사회화 아님" | 비자발적도 **탈사회화** |

#### 일탈

| ❌ 함정 선지 | ✅ 올바른 내용 |
|------------|-------------|
| "결과보다 과정 중시 = 동조주의" | **의례주의** |
| "약물복용 = 의례주의" | **혁신주의** |
| "부상 투혼 = 과소동조" | **과잉동조** (인내 규범 과잉 수용) |
| "약물복용 지속 = 일탈 순기능" | **역기능** (공정성 훼손) |
| "이동방향 = 세대내/세대간" | 이동 **방향**은 수직/수평; 세대는 **시간** 기준 |

#### 미디어

| ❌ 함정 선지 | ✅ 올바른 내용 |
|------------|-------------|
| "핫 미디어 = 경기 속도 빠름" | 빠른 경기 = 쿨 미디어 |
| "보편적 접근권 = 독점 중계권" | 완전 반대 개념 |
| "탁구공 색 변경 = 스포츠가 미디어에 미친 영향" | **미디어가 스포츠에 준 영향** |
| "여성 선수 미디어 = 성과에 더 집중" | **외모·여성성** 부각 (성차별) |

#### 계층·세계화

| ❌ 함정 선지 | ✅ 올바른 내용 |
|------------|-------------|
| "인기·비인기 종목 구분 = 역사성" | **보편성** |
| "개척자형 = 금전적 보상 추구" | **용병형**이 금전 추구; 개척자형은 선교사적 |
| "세계화 → 전통 스포츠 세계적 확대" | **전통 스포츠 소외** |
| "외국인 선수 증가 → 국가 정체성 강화" | **국가 정체성 약화** |
| "스태그플레이션 = 상업화 촉진 조건" | 상업화 **저해** 요인 |

## 섹션 5: 압축 암기 공식

```
【사회화 4단계】
로의(입문) → 통한(변화) → 탈(중단) → 재(복귀)

【머튼 아노미 5유형】
동조=목표O수단O / 혁신=목표O수단X(불법)
의례=목표X수단O / 도피=목표X수단X / 반역=대체

【파슨즈 AGIL】
A(적응)·G(목표달성)·I(통합)·L(잠재적체제유지)

【거트만 근대스포츠 7특징】
세속화·평등성·전문화·합리화·관료화·수량화·기록추구
(암기: 세평전합관수기)

【정치의 스포츠 이용 3방법】
상징(의례) + 동일화(감정이입) + 조작(3S정책)

【스트랭크 국제정치 기능】
베를린=이념선전 / 핑퐁=외교도구 / 뮌헨=갈등표출

【투민 계층 이동 준거】
방향(수직↕·수평↔) + 주체(개인·집단) + 시간(세대내·세대간)

【투민 계층 5특성】
역사성·보편성·고착성·영향성·다양성

【프로스포츠 제도】
드래프트(계약금경쟁방지) + 샐러리캡(연봉한도)
FA(자유계약) + 보류조항(이적방지) + 웨이버(방출공시)

【코클리 4대 윤리규범】
구분짓기·인내·몰입·도전 → 4개 모두 정답

【메기&서덴 노동이주】
용병(돈)·개척자(전파)·정착민(정착)·귀향민(복귀)·유목민(취향)

【미디어 이론 4종】
개인차(욕구충족) + 사회범주(집단) + 사회관계(중요타자) + 문화규범(매체→규범)
```
