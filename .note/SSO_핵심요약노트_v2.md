## 섹션 2: 핵심 노트 (반복출제 문제 기반)

# Chapter 1. 스포츠사회학 개론

## 스포츠의 사회적 기능 (AGIL·순기능) · 🔥🔥🔥

**파슨즈(T. Parsons) AGIL 모형**

| 구성요소        | 영문            | 내용                               |
| --------------- | --------------- | ---------------------------------- |
| 적응            | Adaptation      | 환경에서 자원을 확보하고 체제 유지 |
| 목표성취        | Goal Attainment | 목적 설정 및 달성을 위해 자원 동원 |
| 통합            | Integration     | 구성원 유대 강화, 갈등 조정        |
| 잠재적 유형유지 | Latency         | 사회 가치·규범을 유지하고 전수     |

**스티븐슨·닉슨(Stevenson·Nixon) 구조기능주의 스포츠 기능**

| 기능               | 내용                          |
| ------------------ | ----------------------------- |
| 사회·정서적 기능   | 스트레스 해소, 정서적 안정    |
| 사회통합 기능      | 공동체 의식 강화, 유대감 형성 |
| 사회계층 이동 기능 | 노력에 따른 신분 상승 가능    |

> ⚠️ **주의** **상업주의**는 AGIL 모형 구성요소가 아님 — 매년 등장하는 오답 선지 / **사회갈등 유발**은 구조기능주의가 아닌 **갈등이론** 관점

---

## 스포츠사회학의 정의·연구영역 · 🔥🔥

| 구분                | 내용                                                          |
| ------------------- | ------------------------------------------------------------- |
| 정의                | 스포츠 현장의 사회구조·사회과정을 분석하는 사회학의 하위분야  |
| 거시 영역           | 정치·경제·종교·교육·매스미디어 등 사회제도                    |
| 미시 영역           | 소집단·사회화·일탈·공격성 등 개별 상호작용                    |
| 스포츠심리학과 구분 | 개인의 운동수행능력·심리적 원인 분석 → 스포츠사회학 대상 아님 |

> ⚠️ **주의** **조직**(팀, 연맹 등)은 거시가 아닌 **미시·중범위** 영역 — "거시 영역에 해당하는 것" 물을 때 오답 선지로 자주 등장

---

## 스포츠사회학 이론 · 🔥🔥

| 이론              | 핵심 관점                                        | 스포츠 적용                       |
| ----------------- | ------------------------------------------------ | --------------------------------- |
| 구조기능주의      | 사회는 균형·통합 유기체, 각 부분이 기능 수행     | 스포츠 = 통합·적응·사회화 기능    |
| 갈등이론          | 지배계급이 피지배계급을 착취, 재화의 불평등 분배 | 스포츠 = 자본가의 이데올로기 도구 |
| 상징적 상호작용론 | 미시적 의미 부여·상호작용 중시                   | 낙인, 상황적 일탈 분석            |
| 비판이론          | 구조 모순 지적 + 문화적 주체성·변화 가능성 강조  | 갈등론보다 주체성·저항 중시       |

> ⚠️ **주의** 갈등이론 ≠ 비판이론 — 비판이론은 **문화적 주체성과 변화 가능성**을 더 중시

---

## 미래스포츠 변화 · 🔥🔥

- 과학기술(AI·빅데이터) → 경기 분석·심판 보조 고도화
- 가상현실(VR)·증강현실(AR) → 새로운 스포츠 참여·관람 형태
- 인터넷·SNS → 스포츠 콘텐츠 소비 다변화
- 엘리트 스포츠 ↔ 생활체육 선순환 구조 강화

> ⚠️ **주의** 과학기술 발전이 **스포츠의 공정성·인간성을 훼손**한다는 부정적 측면도 기출됨 — 무조건 긍정적이지 않음

# Chapter 2. 스포츠와 정치

## 정치의 스포츠 이용 · 🔥🔥

| 기제   | 내용                                              | 기출 예시                          |

| 상징화 | 국기·국가 연주·의례로 국가 권위·정체성 부여       | 경기 전 국기에 대한 경례, 국가연주 |

> ⚠️ **주의** **상징화** = 의례적 기호를 통한 국가 권위 부여 / **동일화** = 선수·팀과의 정서적 일체감 — 두 개념 혼동 자주 출제

## 스포츠의 정치적 속성 (에티즌·세이지) · 🔥🔥

| 속성       | 내용                                                   |
| ---------- | ------------------------------------------------------ |
| 보수성     | 현재의 사회 질서·지배적 가치를 유지하려는 성향         |
| 대표성     | 유니폼·마스코트·의례로 집단을 상징하며 충성심 고취     |
| 상호의존성 | 정치와 스포츠가 목적 달성을 위해 서로 돕는 유기적 관계 |
| 권력투쟁   | 스포츠 내외의 영향력·자원 확보를 위한 갈등적 상황      |

> ⚠️ **주의** **대표성** = "특정 집단을 대리·대표"하는 기제 — "상징화"(의례적 상징)와 구분

## 프로스포츠 제도 · 🔥🔥

| 제도                      | 내용                                           | 핵심 구분                                 |
| ------------------------- | ---------------------------------------------- | ----------------------------------------- |
| 보류조항 (Reserve Clause) | 구단이 선수 보류권 독점 → 이적 제한, 몸값 억제 | 선수단 운영비 절감 목적                   |
| 드래프트 (Draft)          | 신인 선수 배분 → 팀 간 전력 균형               | 구단이 지명권 보유                        |
| 샐러리 캡 (Salary Cap)    | **팀 전체** 연봉 총액 상한 → 팀 간 전력 균형   | 개인 최대 연봉 ≠ 샐러리 캡                |
| 최저연봉제                | 저액 선수 생존권 보호                          | 신인 협상력 약화 목적 아님                |
| FA (자유계약)             | 일정 조건 충족 시 모든 구단과 자유 협상        | 선수 권익 보호                            |
| 트레이드 (Trade)          | 구단 간 선수 교환·이적                         | 구단의 권리 — 선수가 일방적으로 요구 불가 |

> ⚠️ **주의** **샐러리 캡** = 팀 전체 연봉 총액 제한 (팀 간 균형) ≠ 선수 개인 연봉 상한선 / **트레이드** = 구단의 권리, 선수의 권리 아님

## 투민(M. Tumin)의 계층 형성과정 · 🔥🔥🔥

| 3    | 평가        | 서열에 사회적 가치 부여 | 우승 시 명성 상승, 선수를 선망의 대상으로 여김 |

## 계층이동 유형 · 🔥🔥🔥

| 기준 | 유형 A                               | 유형 B                                      |
| ---- | ------------------------------------ | ------------------------------------------- |
| 방향 | **수직이동** (위계적 지위 변화 수반) | **수평이동** (지위 변화 없이 소속만 변경)   |
| 시간 | **세대 내 이동** (개인 생애 내 변화) | **세대 간 이동** (부모-자식 세대 간 비교)   |
| 주체 | **개인이동**                         | **집단이동** (사회 전체 변동으로 집단 이동) |

기출 예시 암기:

- 선수 → 코치: 수직이동 + 세대 내 이동 + 개인이동
- 부모보다 높은 수입·명예 획득: **세대 간 수직이동**
- 같은 대우로 다른 팀 이적: **수평이동**
- 2군 → 1군 승격: **수직이동**

> ⚠️ **주의** 수직 = 지위 변화, 수평 = 지위 변화 없음 — 선수→코치는 **수직이동** (자주 틀리는 포인트)

| 이론                              | 핵심 관점                                                | 기제·특징                                 |

| 사회학습이론 (레오나르드·Leonard) | 행동은 상벌(강화)·관찰(모방)로 습득                      | 강화(Reinforcement), 모방(Modeling), 코칭 |
| 역할이론                          | 사회적 지위에 따른 기대 역할을 내면화                    | 역할 기대 → 역할 수행                     |
| 준거집단이론                      | 본보기 집단(실제 소속 or 이상적 집단)의 가치·행동에 영향 | 준거 집단 = 소속 집단 or 동경 집단        |

기출 예시:

- A코치가 상벌로 학생선수 능력 향상 → **강화(사회학습)**
- B선수가 타 선수 타격 자세를 관찰·발전 → **모방(사회학습)**

> ⚠️ **주의** 사회학습이론의 두 기제: 상벌 = **강화**, 관찰 = **모방** — 세트로 암기

# Chapter 7. 스포츠 세계화

### 글로컬리제이션 (로버트슨·Robertson)

- **세계화(Global) + 지역화(Local)**의 합성어
- 세계적 스포츠가 지역 문화와 결합해 새로운 형태로 변용·적응
- 예: LA 다저스가 박찬호 영입 → 한국에서 메이저리그 인기 급상승

---

### 세계화와 민족주의 관계

- 세계화가 심화되어도 민족주의는 오히려 강화되는 경향
- 다국적 기업은 올림픽·월드컵의 민족주의적 열정을 상업적 마케팅 자원으로 활용
- 냉전시대 세계화는 민족주의를 약화시키지 않고 **오히려 이데올로기 대결로 강화**

### 스나이더(E. Snyder)의 사회화 전이 조건

스포츠를 통해 배운 가치·태도가 일반 사회로 전이되기 위한 조건:

- 참여 기간이 길수록 전이 효과 높음
- **자발적** 참여일수록 내면화 깊음
- 참여 강도가 높을수록 영향 큼
- 지지적인 사회적 환경일수록 전이 용이

❌ 스포츠 참여 자체가 자동으로 사회화를 보장하지 않음 — 전이 조건이 충족되어야 함

# Chapter 10. 스포츠와 젠더

### 크로젯(T. Crosset)의 연구 — 스포츠 문화와 성폭력

- 남성 중심 스포츠 문화에서 **폭력이 남성성의 발현**으로 잘못 인식
- 폭력이 여성을 통제하는 효과적 전략이라는 믿음이 일탈을 정당화
- 지역사회가 남성 선수의 폭력에 **경외감**을 갖는 문화 문제
- 여성 선수를 존경·공동체의 일원이 아닌 **통제 대상**으로 간주

❌ "지역사회가 남성 선수의 폭력에 철저히 처벌" → 오히려 **묵인·경외**하는 문화가 문제

### 미디어의 성차별 이데올로기

- 여성 스포츠 종목 중계 비중 낮음
- 여성 선수를 경기력보다 **외모·성적 매력**으로 묘사
- 남성 스포츠 = 정상, 여성 스포츠 = 부수적 이미지 고착화
- 미디어가 성 역할 고정관념 강화 및 재생산

❌ "미디어는 스포츠의 성차별 이데올로기를 극복한다" → 오히려 미디어가 성차별 이데올로기를 **생산·강화**
