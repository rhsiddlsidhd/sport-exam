# Chapter 1. 스포츠윤리 개요

## 스포츠윤리 개념·목적·역할 · 🔥🔥🔥

| 구분                  | 내용                                                                         |
| --------------------- | ---------------------------------------------------------------------------- |
| 정의                  | 스포츠 상황에서 발생하는 도덕 문제를 탐구하는 실천 윤리학의 한 분야          |
| 목적                  | 스포츠 참가자의 도덕적 판단력 향상, 스포츠의 가치 보호                       |
| 역할                  | 스포츠에서 발생하는 폭력·차별·도핑 등의 문제를 윤리적으로 해결하는 규범 제공 |
| 스포츠윤리 특징(2024) | 실천성·구체성·맥락 의존성 — 추상적 이론보다 **실제 상황 적용**을 강조        |

> ⚠️ **주의** 스포츠윤리는 "규칙만 지키면 된다"는 관점을 거부한다. 규칙 이상의 도덕적 태도를 요구한다는 점이 자주 선지에 등장한다.

## 사실판단 vs 가치판단 · 🔥🔥🔥

| 구분            | 사실판단                          | 가치판단                          |
| --------------- | --------------------------------- | --------------------------------- |
| 성격            | 참·거짓 검증 가능 (경험적)        | 옳고 그름·좋고 나쁨 평가 (규범적) |
| 예시            | "이 선수는 100m를 9.8초에 달렸다" | "이 선수는 훌륭한 선수다"         |
| 윤리학과의 관계 | 윤리학의 대상이 아님              | 윤리학의 핵심 대상                |

> ⚠️ **주의** "사실판단은 윤리학의 대상"이라는 선지는 오답이다. 가치판단이 윤리학의 대상이다.

# Chapter 2. 윤리이론

## 의무론(칸트·선의지·정언명령) · 🔥🔥🔥

| 개념                               | 내용                                                                  |
| ---------------------------------- | --------------------------------------------------------------------- |
| 선의지(Guter Wille)                | 결과·경향성과 무관하게 의무 자체를 위해 행동하는 의지. 그 자체로 선함 |
| 정언명령(Kategorischer Imperativ)  | "네 행위의 준칙이 동시에 보편적 법칙이 될 수 있도록 행동하라"         |
| 의무에서 나온 행위(aus Pflicht)    | 도덕적 가치 있음 — 의무를 **이유로** 행동                             |
| 의무에 합치하는 행위(pflichtmäßig) | 도덕적 가치 없음 — 결과적으로 의무에 맞지만 다른 동기(이익 등)로 행동 |
| 의무론의 난점                      | 결과를 전혀 고려하지 않으면 현실에서 극단적 상황 발생 가능            |

> ⚠️ **주의** "의무에 합치하는 행위"도 도덕적 가치가 있다는 선지는 오답이다. 반드시 "의무에서 나온" 행위여야 한다. 칸트는 결과가 아닌 **동기**를 본다.

## 공리주의·결과론 · 🔥🔥🔥

| 개념                       | 내용                                                        |
| -------------------------- | ----------------------------------------------------------- |
| 기본 원리                  | 최대 다수의 최대 행복                                       |
| 행위 공리주의              | 개별 행위의 결과가 최대 행복을 낳는가로 판단                |
| 규칙 공리주의              | 일반적으로 최대 행복을 산출하는 규칙을 따름                 |
| 결과론의 제한점(2019·2020) | 소수의 희생 정당화, 개인 권리 침해 가능, 행복 계산의 어려움 |

> ⚠️ **주의** 공리주의는 **결과**만을 도덕 판단 기준으로 삼는다. "동기"나 "의도"는 고려하지 않는다. 이것이 의무론과의 핵심 차이.

## 덕 윤리 · 🔥🔥

| 학자명(영문)              | 개념                 | 핵심 키워드                                       |
| ------------------------- | -------------------- | ------------------------------------------------- |
| 아리스토텔레스(Aristotle) | 아레테(탁월성·덕)    | 습관·실천으로 형성되는 덕, 중용(中庸)             |
| 맥킨타이어(MacIntyre)     | 실천(Practice) 내 덕 | 덕은 특정 사회적 실천 공동체 안에서 획득됨 (2024) |

> ⚠️ **주의** 덕 윤리는 "어떤 행위를 해야 하는가"보다 "어떤 사람이 되어야 하는가"(행위자)를 묻는다. 행위 규칙 중심의 의무론·공리주의와 구별하자.

# Chapter 3. 경쟁과 페어플레이

## 경기에 참여하는 목적 · 🔥🔥🔥

| 개념          | 원어     | 의미                                                                        |
| ------------- | -------- | --------------------------------------------------------------------------- |
| 아곤(agon)    | 그리스어 | 경쟁, 승리를 향한 겨룸                                                      |
| 아레테(arete) | 그리스어 | 탁월성, 자기 능력의 최대 발휘                                               |
| 관계          | —        | 대립이 아닌 **상호보완** — 공정한 경쟁(아곤) 속에서 탁월성(아레테)이 드러남 |

> ⚠️ **주의** "아곤과 아레테는 서로 충돌한다"는 선지는 오답. 진정한 경쟁(아곤)은 탁월성(아레테) 발현의 기회다.

## 철학 관련 용어 · 🔥🔥🔥

| 개념          | 설명                                            | 기출 예시                          | 학자                              |
| ------------- | ----------------------------------------------- | ---------------------------------- | --------------------------------- |
| 에토스(ethos) | 선수들이 암묵적으로 공유하는 비공식적 규범·관행 | 성문화되지 않은 도덕적 기대 (2024) | 아리스토텔레스의 설득의 핵심 요소 |

> ⚠️ **주의** 에토스는 **공식 규칙이 아니다**. 선수집단이 비공식적으로 공유하는 관행이다. 형식주의(규칙 만능론)와 대비되는 개념.

# Chapter 4. 스포츠와 정의

## 정의 유형(평균적·절차적·분배적) · 🔥🔥🔥

| 유형        | 원리                                  | 스포츠 적용 예                     |
| ----------- | ------------------------------------- | ---------------------------------- |
| 평균적 정의 | 같은 것은 같게, 차별 없는 동등 대우   | 동일 규칙을 모든 선수에게 적용     |
| 절차적 정의 | 공정한 과정·절차를 거치면 결과도 정의 | 공정한 심판 진행, 추첨에 의한 대진 |
| 분배적 정의 | 기여·능력·필요에 비례한 배분          | 연봉, 시상, 포상의 차등            |

> ⚠️ **주의** "절차적 정의는 결과의 공정성을 보장한다"는 선지 — 반은 맞고 반은 틀리다. 절차적 정의는 **결과가 아닌 과정**의 공정성에 주목한다.

# Chapter 5. 도핑과 과학기술

## 약물도핑(WADA·엄격책임·TUE) · 🔥🔥🔥

| 개념                       | 내용                                                               |
| -------------------------- | ------------------------------------------------------------------ |
| WADA(세계반도핑기구)       | 금지약물·방법 목록 관리, 검사 기준 수립                            |
| 엄격책임(strict liability) | 선수의 의도·과실 여부에 관계없이 금지약물이 체내에서 검출되면 책임 |

> ⚠️ **주의** 엄격책임은 "몰랐어도 책임진다"는 원칙이다. "의도가 없으면 처벌 안 된다"는 선지는 오답.

> ⚠️ **주의** 기술도핑의 핵심 쟁점은 **공정성(fairness)** 훼손 여부다. 기술 자체의 선·악이 아니라 경쟁의 동등한 조건을 깨는지가 기준.

### 형식주의 vs 비형식주의
