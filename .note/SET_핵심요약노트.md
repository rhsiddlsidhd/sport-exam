> **강사 한마디**: 스포츠윤리는 "이론 이름 ↔ 특징 ↔ 난점"을 세트로 외워야 한다. 매년 의무론·공리주의·덕윤리 구분 문제가 반드시 출제되고, 정의 유형(평균/분배/절차) 매칭도 빠지지 않는다. 학자 이름과 이론을 헷갈리게 섞어 놓은 선지가 함정의 핵심.

## 섹션 2: 토픽별 핵심 개념

### 1위. 윤리이론 (의무론 / 공리주의 / 덕윤리)

| 이론 | 핵심 원리 | 스포츠 적용 | 난점 |
|------|-----------|-------------|------|
| **의무론 (칸트)** | 결과 무관, 도덕규칙·의무 자체가 기준. 정언명령. 선의지 강조. | VAR처럼 규칙 엄격 적용, 결과와 무관하게 반칙은 반칙 | 도덕규칙 간 충돌 시 실질 지침 부재 |
| **공리주의 (벤담/밀)** | 최대다수의 최대행복. 결과·유용성 기준. | 스포츠 정책, 규칙 개정 시 전체 이익 계산 | 사회 전체 이익 고려 못하는 경우 발생. 정의 문제 소홀 |
| **덕윤리** | '무엇을 해야 하는가'보다 '어떤 사람이 되어야 하는가'. 품성·인성 강조 | 선수의 내면적 품성, 스포츠맨십 | '올바른 행위'보다 행위자 인성에 집중 → 규범 불명확 |
| **배려윤리 (나딩스)** | 관계·감정·배려 중심 | 부상 선수 돕기, 팀 동료 배려 | — |
| **사회윤리 (니부어)** | 개인 윤리 ≠ 집단 윤리. 도덕적 개인도 비도덕적 집단에 동화 가능 | 팀 분위기에 의한 반칙 동화 | — |
| **책임윤리 (요나스/베버)** | 미래 결과·세대에 대한 책임 | 환경 파괴 스포츠 시설 자제 | — |

**칸트 의무론 세부 구분** (2025 신출제):
- **의무에서 나온(aus Pflicht) 행위**: 의무 자체가 동기 → 도덕적 가치 있음
- **의무에 합치하는(pflichtmäßig) 행위**: 행위 결과는 맞지만 동기가 의무 아님 → 도덕적 가치 없음

**결과론 vs 의무론 구분 포인트**:
- 결과론: "행위로 발생하는 결과에 따라 옳고 그름 결정"
- 의무론: "결과에 상관없이 절대적 도덕규칙에 따라 판단"
- 의무론 해당: 선의지, 정언명령, 도덕규칙 준수, 목적은 수단 정당화 불가
- 공리 해당: 유용성 원리, 공평성 원리, 최대다수 최대행복

### 2위. 스포츠와 인권·차별

| 유형 | 핵심 내용 |
|------|-----------|
| **인종차별** | 피부색에 따른 능력 차이는 절대적이지 않다. 흑인 특유의 능력 언급 = 차별 |
| **성차별** | 여성 스포츠 참여 제한, 종목 성별 고정화, 성의 상품화 모두 차별 |

| **장애차별** | 비장애인과 동일 훈련량 강요 = 차별. 접근성·시설 확충 필요 |
| **타이틀 나인(Title Ⅸ)** | 미국 교육법 개정 → 여성 스포츠 참여 기회 확대 (2025 신출제) |
| **1968 멕시코올림픽** | 토미 스미스·존 카롤로스 → 검은 장갑·양말 = **인종차별** 저항 |

### 3위. 폭력·반칙·도핑

**반칙 유형 (롤랜드(Loland) 분류)** (2025 신출제):
- 규제적 규칙(regulative rules) 위반 vs 구성적 규칙(constitutive rules) 위반
- 태권도 전자호구 조작 = 규제적 위반 / 수영 화상자국 은폐 전신수영복 = 해당 없음

**도핑**:
- WADA 금지 방법: 물리적 조작, 화학적 조작, 유전자 도핑 (침술은 해당 없음)
- 치료목적 사용 물질도 선수 책임 (면책 불가)
- 기술도핑(전신수영복 등): 최첨단 장비는 공정성 저해 → FINA 금지 근거

**폭력 유형 (학자별)**:
| 학자 | 개념 | 스포츠 적용 |
|------|------|-------------|
| 푸코(Foucault) | 규율과 권력 | 위계적 권력 관계 → 폭력으로 변질 |
| 아렌트(Arendt) | 악의 평범성 | 잘못된 관행 복종·반복 |
| 아리스토텔레스 | 분노 | 내면 분노 → 폭력 악순환 |
| 지라르(Girard) | 모방적 경쟁 | 선수 폭력 모방 (홉스 아님!) |

### 4위. 심판윤리

**심판의 덕목**:
| 덕목 | 설명 |
|------|------|
| **공정성** | 선수 이익 동등 대우, 중립성, 개인 감정 배제 |
| **전문성** | 규정 숙지, 오랜 경험과 훈련, 정확한 판정 능력 |
| **자율성** | 외부 지시·간섭 단호히 거부 |
| **청렴성** | 탐욕 없음, 심판으로서 품위 유지 |

- **심판의 개인윤리** vs **사회윤리**: 사회윤리는 협회·종목단체의 도덕성과 연결
- "판정 신뢰성을 높이는 제도 도입" = 개인윤리적 덕목이 **아님** (제도는 사회윤리 영역)
- "이익동등 고려 원칙으로 약한 팀에 유리한 판정" = **오답** (이익동등 = 동등 대우, 편파 아님)

### 6위. 페어플레이·스포츠맨십

- **페어플레이**: 규칙 준수 + 상대 존중 + 근대 시민 도덕규범
  - 규칙 준수로서 페어플레이 = 행위 요구와 제재 의미 (맞는 선지)
  - "선수 개인의 의도나 목적에 따라 변화" = **오답** (페어플레이는 객관적 기준)
- **스포츠맨십**: 합규칙성을 **넘는** 적극적 도덕적 마음가짐 (맞는 선지)
- **에토스(Ethos)**: 규칙 외적 비공식 관행. 부상 선수에게 공 차주기, 모자 벗어 사과 등
  - 배구 블로킹 시 훈련대로 네트 안 닿게 주의 = **에토스 실천 아님** (규칙 준수일 뿐)
- **형식주의**: 규칙 위반 자체로 비윤리적
- **비형식주의**: 규칙 위반도 맥락·목적에 따라 허용 가능

### 9위. 도덕성 발달 (레스트·뒤르켐)

**레스트(Rest) 도덕성 4구성요소**:

| 번호 | 구성요소 | 스포츠 교육 적용 |
|------|----------|-----------------|
| 1 | 도덕적 민감성(moral sensitivity) | 도덕적 딜레마를 지각하게 하는 것 |
| 2 | 도덕적 판단력(moral judgement) | 옳고 그름을 판단하게 하는 것 |
| **3** | **도덕적 동기화(moral motivation)** | **다른 가치보다 정정당당한 경기에 가치 두게 하는 것** |
| 4 | 도덕적 품성화(moral character) | 강한 의지·용기·인내 등 품성 갖게 하는 것 |

- 3번째 = **도덕적 동기화** (매년 출제 핵심, 동기화↔판단력 혼동 주의)

**뒤르켐(Durkheim) 도덕교육론** (2025 신출제):
- 규율·집단 소속감·자율성 3요소 → 스포츠윤리 교육에 적용

## 섹션 3: 함정 선지 총정리

### 윤리이론 혼동

❌ "절묘한 기술로서 '좋은 패스'는 도덕적 선(善)으로 해석된다."
→ 기술적 의미의 '좋음'은 도덕적 선이 아님

❌ "공리적－칸트(I. Kant)"
→ 칸트는 의무론(정언명령). 공리주의는 벤담(Bentham)·밀(Mill)

❌ "덕윤리는 '무엇이 올바른 행위인지'를 판단하는 데 더 주목한다."
→ 덕윤리는 '어떤 사람이 되어야 하는가(행위자 인성)'에 주목. 행위 기준은 의무론·공리주의

❌ "의무론의 난점: 행위의 결과를 즉각 산출하기 어려울 경우 지침 제시 못함"
→ 이것은 공리주의의 난점. 의무론 난점 = 도덕규칙 간 충돌 시 해결 어려움

❌ "A선수(마라톤 쓰러진 선수 도움)의 입장 난점: 인간을 항상 목적으로 대해야 한다"
→ 이는 의무론의 원칙(난점 아님). 정답 난점 = 공리주의(사회 이익·손해 고려)

❌ "VAR 반대 입장 = 공리주의"
→ 규칙 엄격 적용(결과 무관) = 의무주의. 최대다수 최대행복 계산 = 공리주의

### 심판윤리 혼동

❌ "㉠-공정성 / ㉡-자율성" (심판 덕목 매칭)
→ 중립성·감정배제 = 공정성 / 오랜 경험·정확 판정 = **전문성**. 자율성은 외부 간섭 거부

❌ "이익동등 고려 원칙에 따라 약한 팀에 유리한 판정 가능"
→ 이익동등 = 동등하게 대우. 특정 팀 편파 판정은 절대 불가

❌ "판정의 신뢰성을 높이는 제도 도입 = 개인윤리적 덕목"
→ 제도는 사회윤리·조직윤리 영역. 개인 덕목 아님

### 정의 유형 혼동

❌ "스포츠 바우처 = 기회균등의 원칙"
→ 약자에게 특별 지원 = **차등의 원칙**. 기회균등은 동등한 기회 제공

❌ "동전 던지기 선후공 결정 = 분배적 정의"
→ 공정한 과정·절차 = **절차적 정의**

### 기타 혼동

❌ "홉스(T. Hobbes)의 폭력론 = 모방적 경쟁"
→ 모방적 경쟁은 **지라르(Girard)**. 홉스는 자연 상태·자기 보존 본능

❌ "베르크(Berque)의 환경윤리 = 자연중심주의"
→ 베르크는 인간 사용 가치에 비례해 자연 평가 → 인간중심주의에 가까움

❌ "스포츠맨십은 합규칙성의 준수를 의미한다"
→ 스포츠맨십은 합규칙성을 **넘는** 적극적 도덕적 마음가짐

❌ "페어플레이는 선수 개인의 의도나 목적에 따라 변화하는 도덕적 행위"
→ 페어플레이는 객관적 기준. 개인 의도에 따라 변화하지 않음

## 섹션 4: 최근 3년 신출제 경향

### 2023년
- **스포츠윤리센터** 법적 근거 (국민체육진흥법 제18조의3) 첫 출제
- **유전자 도핑** 반대 이유 (ITEMIZED)
- **지속 가능한 발전** 관점 출제
- **형식주의 vs 비형식주의** 명칭 직접 매칭 (승부조작 시나리오)
- **FINA 기술도핑 금지** 사례 분석

### 2024년
- **칸트 의무론 vs 공리주의** 구분 문제 급증 (PASSAGE 유형 7문항)
- **VAR** 기술 적용을 윤리이론으로 분석
- **행위 공리주의** 용어 직접 출제
- **롤랜드(Loland) 반칙 유형 분류** 첫 출제

### 2025년
- **칸트 의무론 세부**: "의무에서 나온 행위 vs 의무에 합치하는 행위" 독일어 원어 포함
- **타이틀 나인(Title Ⅸ)** 미국 교육법 출제
- **레건(Regan) 동물권리론** 첫 출제 (싱어와 구분)
- **뒤르켐(Durkheim) 도덕교육론** 첫 출제
- **부올레(Vuolle) 스포츠 환경 분류** 첫 출제
- **함무라비 탈리오 법칙** 정의론 문제 출제
- **트랜스젠더 스포츠 참여** 심화 (IOC 2016 지침)

**트렌드 요약**: 2023년까지 이론 이름 매칭이 주였다면, 2024~2025년은 구체적 사례에서 이론을 역추론하는 응용형으로 심화되고 있다. 칸트 의무론의 세부 개념과 신규 학자(레건, 뒤르켐, 롤랜드) 출제가 증가 추세.
