# 한국체육사(HIS) 핵심 요약 노트
### 2급 생활스포츠지도사 필기 | 2019~2025 기출 분석 기반 (140문제)

> 강사 한마디: HIS는 "순서 암기"와 "혼동 함정"이 핵심이다. 개화기가 전체의 18%를 차지하고, 올림픽 연표·정부별 정책이 각각 16%다. 틀리는 문제 패턴은 딱 3가지 — 조선체육회 vs 조선체육협회, 태릉선수촌 건립 주체, 전통/근대 체육 기준점. 이 세 가지만 잡아도 5점이 올라간다.

### 3위. 조선시대 무예 제도·기관·무예서

**훈련원(訓鍊院)**
- 역할: 무예 훈련+병서 습득+군사 **시재(試才)** 담당
- ❌ 친위부대 X / 성리학 교육 X (성리학 = 성균관·향교)

**무과제도**
- 3단계: **초시(예비) → 복시(본시험) → 전시(국왕 앞 등위 결정)**
- 시험 내용: **무예 실기 + 강서(병법 이론)**
- 식년시 정원: **28명** (정원 없다는 것 X)
- 주관: **훈련원+병조** (예조 X)
- 부정기 시험: 증광시·별시·정시 등

**궁술(弓術)**
- 육예(六藝) 중 → **사(射)** (어御 X → 어御는 마술/말타기!)
- 학사사상(學射思想): 인격 수양·심신 단련
- 의례: 대사례(大射禮, 왕-신하), 향사례(鄕射禮, 선비)
- ❌ 불국토사상과 연결 X → 불국토사상은 신라 화랑도!

**활인심방(活人心方)**
- 저술자: **이황(李滉·퇴계)** ← 이이(율곡) X!
- 근거: 중국 문헌 『활인심(活人心)』
- 내용: 도인법(신체 스트레칭·단련), 사계양생가(계절별 호흡), 활인심서(욕망 절제)

### 4위. 일제강점기 체육

**조선체육회 vs 조선체육협회 완벽 정리**

| 성격 | **일제 관변단체** | **민족 체육단체** |
| 후원 | 총독부 기관지 | **동아일보사** |

| 주요 사업 | 조선신궁경기대회(관제) | 1920년 **전조선야구대회**(→전국체전 효시!) |

- **서상천**: 역도 도입, 조선체력증진법연구회 설립, 『현대체력증진법』 발간

### 8위. 사료·사관·시대구분

**사료 종류 (매년 1문제)**

| 사료 유형 | 예시 | 함정 |
|-----------|------|------|
| 물적 사료 | 유물·유적·벽화·각저총 | 트로피·메달·경기복장 = **물적 사료** (구전 X!) |
| 문헌 사료 | 무예도보통지, 회고록, 공문서 | 고구려 무용총 수렵도 = **물적** (문헌 X) |
| 구전 사료 | 민요·전설·설화 | — |
| 구술 사료 | 생존자 증언·회고담 | — |

**사관(史觀) vs 사료(史料) 구분**
- 사관 = 역사를 해석하는 **관점·견해** (유물사관, 관념사관, 진보사관, 순환사관)
- 사료 = 과거 사실을 담은 **구체적 자료**
- 체육사 ≠ 스포츠 윤리학 (도덕적 가치 판단은 체육사 영역 X)

**나현성 시대구분**
```
전통체육 → [갑오경장 1894] → 근대체육 → [광복 1945] → 현대체육
```

## 섹션 3: 함정 선지 총정리

❌ `트로피·메달·경기복장 등은 구전 사료이다` → 형태가 있는 **물적 사료**

❌ `동맹은 부여의 제천행사이다` → 동맹 = **고구려**, 부여 = **영고**

❌ `화랑도는 법흥왕 때 체계화되었다` → **진흥왕** 때 체계화

❌ `화랑도의 도의교육 근간은 삼강오륜이다` → **세속오계** (삼강오륜은 조선 성리학)

❌ `궁술은 육예 중 어(御)에 해당한다` → 궁술 = **사(射)**, 어(御) = **마술**

❌ `활인심방은 이이(율곡)가 저술하였다` → **이황(퇴계)** 저술

❌ `무예도보통지는 고려시대에 편찬되었다` → **조선 정조(1790)** 때 편찬

❌ `훈련원은 성리학 교육을 담당하였다` → 성리학 = 성균관·향교 / 훈련원 = **무예+병서**

❌ `훈련원은 국왕의 친위부대였다` → 훈련원 = 행정·교육 기관, 친위부대 = 내금위 등

❌ `사정(射亭)에서 병서 강습과 마상 무예 훈련을 했다` → 사정 = **활쏘기 전용**, 병서/마상 = 훈련원

❌ `조선체육회는 조선체육협회보다 먼저 창립되었다` → 협회(1914) > 체육회(1920)

❌ `조선체육회는 조선신궁경기대회를 개최했다` → 신궁경기대회 = **일제 관제 대회**

❌ `개화기 최초 운동회는 일본인 학교 주관이었다` → **영어학교** 주관, 화류회(1896)

❌ `운동회 초기 종목은 야구·농구였다` → 육상·병식체조 위주 (구기 X)

❌ `관서체육회는 개화기에 설립되었다` → **일제강점기** (1920년대, 평양)

❌ `대한국민체육회(1907)는 일제강점기 단체이다` → **개화기** 단체

❌ `태릉선수촌은 전두환 정부가 건립하였다` → **박정희 정부(1966)** 건립

❌ `국군체육부대(상무)는 박정희 정부가 창설하였다` → **전두환 정부** 창설

❌ `체육부는 박정희 정부 때 신설되었다` → **전두환 정부(1982)** 신설

❌ `호돌이 계획은 서울 올림픽 대비를 위해 수립되었다` → 올림픽 **이후(1990)** 생활체육 진흥

❌ `1991 남북단일팀은 서울 올림픽 중 남북회담 계기로 구성되었다` → **1990년 남북체육장관회담** 계기 (북한은 서울올림픽 보이콧)

❌ `2018 평창은 하계 아시아경기대회였다` → **동계 올림픽**

❌ `1948년 한국이 처음 참가한 올림픽은 런던 하계이다` → **생모리츠 동계(1948.1)** 가 먼저!

❌ `황국신민체조는 유희 중심의 체조 원리로 교육되었다` → 군국주의·세뇌 목적, 유희 X

❌ `격구는 서민들의 오락 활동으로 급속히 확산되었다` → 말·장비 필요 → **지배층** 전용

❌ `수박은 응방도감에서 관장하였다` → 응방도감 = **매사냥(방응)** 관장
