## 에너지 시스템 · 🔥🔥🔥

### [비교형] 3대 에너지 시스템

| 구분 | ATP-PC 시스템 | 무산소 해당과정 | 유산소 시스템 |
|------|--------------|----------------|--------------|
| 지속 시간 | 0~10초 | 10초~2분 | 2분 이상 |
| 강도 | 초고강도 | 고강도 | 저~중강도 |
| 기질 | 크레아틴인산(PCr) | 글루코스 / 글리코겐 | 글루코스, 지방산, 아미노산 |
| 장소 | 세포질 | 세포질 | **미토콘드리아** (크렙스 + 전자전달계) |
| 생성 ATP | 소량 (즉각) | 글루코스 → **2 ATP**, 글리코겐 → 3 ATP | **32 ATP** (글루코스 1분자) |
| 최종 산물 | 크레아틴 + 인산 | 피루브산 / **젖산** | CO₂ + H₂O |
| ATP 합성률 | 가장 빠름 | 빠름 | **가장 느림** |
| 기출 종목 예 | 역도, 높이뛰기, 20m 달리기 | 400m 질주(약 60초) | 마라톤, 장거리 |

### [개념-설명-예시형] 호흡교환율(RER)

| RER 값 | 주 에너지원 | 비고 |
|--------|------------|------|
| 1.0 | 탄수화물 100% | 고강도 운동 |
| 0.8 | 지방 + 탄수화물 혼합 | 중강도 |
| 0.7 | 지방 위주 | 저강도, 장시간 |

- RER = VCO₂ / VO₂ (질소가 아닌 이산화탄소 / 산소 비율)
- 저강도 장시간 운동 시 시간이 지날수록 RER이 **감소**하고 지방 이용 비중이 높아짐
- 운동 강도가 높을수록 주 연료는 **근글리코겐**

> ⚠️ **주의** 유산소 시스템의 ATP 합성률은 가장 '느림'이지, 생성량이 적은 것이 아님. 총 ATP 생성량은 가장 많다. / RER에서 '질소'가 나오면 오답 — 질소는 단백질 지표이고 RER은 탄수화물·지방 비율 추정에 사용한다.

## 근섬유 유형 · 동원 순서 · 🔥🔥🔥

### [비교형] 지근(Type I) vs 속근(Type II)

| 구분 | 지근 (Type I) | 속근 (Type IIa) | 속근 (Type IIx) |
|------|-------------|----------------|----------------|
| 색상 | 적색 | 중간 | 백색 |
| 수축 속도 | 느림 | 빠름 | 가장 빠름 |
| 피로 저항 | **높음** | 중간 | **낮음** |
| 미토콘드리아 | 많음 | 중간 | 적음 |
| 모세혈관 | 많음 | 중간 | 적음 |
| 마이오글로빈 | 많음 | 중간 | 적음 |
| 근형질세망 | 덜 발달 | - | **잘 발달** |
| 마이오신 ATPase | 느림 | 빠름 | 가장 빠름 |
| 운동뉴런 크기 | **작음** | 중간 | **큼** |
| 칼슘이온 방출 | 느림 | - | **빠름** |
| 주 에너지계 | 유산소 | 혼합 | 무산소 |

### [단계-순서형] 운동단위 동원 순서 (크기의 원리)

| 순서 | 섬유 유형 | 동원 강도 |
|------|----------|----------|
| 1st | **Type I (지근)** | 저강도 (VO₂max ~30% 이하) |
| 2nd | **Type IIa** | 중강도 |
| 3rd | **Type IIx** | 고강도 |

- 운동단위 = **운동뉴런** + 그 뉴런이 지배하는 근섬유들 (감각뉴런이 아님)
- 동원 단위 수 증가 또는 발화 빈도 증가 → 수축력 증가

> ⚠️ **주의** 저강도 운동 시 가장 먼저 동원되는 것은 Type I(지근), Type IIx는 마지막. / 속근의 피로 저항은 **낮음**이 정답 — '높음'이 오답 선지로 자주 등장.

## 근수축 기전 · 🔥🔥🔥

### [단계-순서형] 흥분-수축 결합 (E-C Coupling)

| 단계 | 내용 |
|------|------|
| 1 | 신경근 접합부(NMJ)에서 **아세틸콜린(ACh)** 방출 |
| 2 | 근섬유막 활동전위 발생 → 가로세관(T-tubule) 전달 |
| 3 | **근형질세망(SR)**에서 **칼슘이온(Ca²⁺)** 방출 |
| 4 | Ca²⁺가 액틴의 **트로포닌**과 결합 |
| 5 | **트로포마이오신**이 결합부위에서 비켜남 |
| 6 | 마이오신 머리가 액틴과 결합 → 파워 스트로크(활주) |
| 7 | ATP 소비 → 마이오신 분리 → 이완 또는 재수축 |

### [목록형] 근절(Sarcomere) 구조 수축 시 변화

- **A 밴드**: **변하지 않음** (마이오신 전체 길이)
- **I 밴드**: **짧아짐** (액틴만 있는 구간)
- **H 구역**: **짧아짐** (겹침 증가)
- **근절 전체 길이**: **짧아짐**
- **액틴·마이오신 단백질 자체 길이**: 변하지 않음 (필라멘트가 짧아지는 것이 아니라 겹쳐지는 것)

> ⚠️ **주의** 수축 시 A밴드는 변하지 않음이 정답. / 신경근 접합부에서 분비되는 신경전달물질은 아세틸콜린 — 에피네프린이 오답으로 자주 등장.

## 수축 형태 · 🔥🔥🔥

### [비교형] 수축 형태 3종 비교

| 구분 | 단축성 (Concentric) | 신장성 (Eccentric) | 등척성 (Isometric) |
|------|-------------------|------------------|------------------|
| 근육 길이 | 짧아짐 | 길어짐 | 변하지 않음 |
| 동적/정적 | 동적 | **동적** | 정적 |
| 속도-힘 관계 | 속도↑ → 힘**↓** | 속도↑ → 힘**↑** | 속도=0 → 파워=0 |
| 발생 힘 크기 | 가장 작음 | **가장 큼** | 중간 |
| 예시 | 턱걸이 올라갈 때 | 턱걸이 내려갈 때 | 자세 유지 |
| DOMS | 드묾 | **주된 원인** | 드묾 |

- 근파워 = 힘 × 수축 **속도** (수축시간이 아님)
- 등척성 수축 시 속도=0이므로 파워=0

> ⚠️ **주의** 신장성 수축은 "정적 수축"이 아닌 **동적 수축** — '정적'이 오답으로 출제됨. / 신장성 수축의 힘 크기는 모든 속도에서 단축성보다 크다.

## 1회 박출량 · 심박출량 · 🔥🔥🔥

### [개념-설명-예시형] 핵심 공식

| 개념 | 공식 / 내용 |
|------|------------|
| **1회 박출량 (SV)** | SV = **EDV - ESV** (확장기말 - 수축기말 용적) |
| **심박출량 (Q)** | Q = SV × HR |
| SV 증가 조건 | 전부하(EDV)↑ / 심실 수축력↑ / 후부하(MAP)↓ |
| SV 감소 조건 | 심박수 **과도 증가** → 충만 시간 부족 → SV↓ |

### [비교형] 안정 시 vs 유산소 훈련 후

| 지표 | 훈련 전 | 훈련 후 |
|------|--------|--------|
| 안정시 HR | 보통 | **감소** |
| 안정시 SV | 보통 | **증가** |
| 최대 SV | 보통 | **증가** |
| 최대 HR | 보통 | **변화 없음** |
| VO₂max | 보통 | **증가** |
| 동정맥 산소차 | 보통 | **증가** |

### [목록형] 혈류 재분배

- 운동 중 활동 근육(골격근) → 혈류 집중 (세동맥 확장)
- 내장기관, 신장 → 혈류 감소 (혈관 수축)
- 안정 시 골격근 혈류 비율 약 15~20% → 최대 운동 시 **80~85%** 이상

> ⚠️ **주의** SV 공식은 EDV-ESV (확장기말 - 수축기말). "수축기말 - 확장기말"로 뒤집어 출제. / 유산소 훈련으로 최대 심박수는 변하지 않는다 (증가가 오답).

## 트레이닝 적응 (유산소 / 무산소) · 🔥🔥🔥

### [비교형] 유산소 vs 무산소 트레이닝 적응

| 항목 | 유산소 (지구성) | 무산소 (저항/스프린트) |
|------|--------------|----------------------|
| 미토콘드리아 | 수·크기 **증가** | 상대적 밀도 감소 |
| 모세혈관 밀도 | **증가** | 큰 변화 없음 |
| 마이오글로빈 | **증가** | 큰 변화 없음 |
| 근섬유 변화 | IIx → IIa 전환 | 속근(Type II) **비대** |
| 해당 효소(PFK 등) | 큰 변화 없음 | **활성 증가** |
| PCr 저장량 | 큰 변화 없음 | **증가** |
| 수축 속도 | 큰 변화 없음 | **증가** |
| 동정맥 산소차 | **증가** | 큰 변화 없음 |
| 안정시 SV | **증가** | 큰 변화 없음 |

### [목록형] 자주 나오는 함정 정리

- 유산소 트레이닝: 지근 → 속근 전환 **없음** (반대 방향인 IIx → IIa는 가능)
- 유산소 트레이닝: Type II 섬유의 현저한 **크기 증가 없음** (저항성 트레이닝의 효과)
- 저항 트레이닝: 근섬유의 **수(개수) 증가 없음** — 성인에서 근비대(크기 증가)만 발생
- 저항 트레이닝 후 신경근 접합부 크기는 **증가**, 시냅스 소포 수는 **증가**

### [목록형] 트레이닝 5원리 (빈출)

- **과부하** : 일상 수준 이상의 자극 (빈도·강도·시간 증가)
- **특이성** : 훈련 효과는 사용 근육·에너지 시스템에 국한
- **가역성** : 훈련 중단 시 적응 소멸

> ⚠️ **주의** 유산소 훈련으로 속근이 지근으로 바뀐다는 선지는 오답. / 저항 트레이닝은 미토콘드리아 밀도를 절대적으로 줄이지 않으나, 근비대로 인해 상대적 밀도는 낮아질 수 있다.

## 호르몬 · 내분비 · 🔥🔥🔥

### [개념-설명-예시형] 핵심 호르몬 일람

| 호르몬 | 분비 기관 | 주요 작용 | 운동 중 변화 |
|--------|---------|----------|------------|
| **에피네프린** | **부신수질** | 글리코겐 분해, 심박수↑, 지방분해↑ | **증가** |
| 노르에피네프린 | 교감신경 말단 / 부신수질 | 혈관수축, 혈압↑ | 증가 |
| **인슐린** | 췌장 **베타(β)세포** | 혈당↓, 지방합성↑, 지방분해 **억제** | **감소** |
| **글루카곤** | 췌장 **알파(α)세포** | 간 글리코겐 분해, 혈당↑ | 증가 |
| **코티솔** | **부신피질** | 당신생, 유리지방산 동원↑, 단백질 분해 | 증가 |
| **알도스테론** | **부신피질** | Na⁺ 재흡수↑, 수분 손실 억제 | 증가 |
| **ADH** | **뇌하수체 후엽** | 수분 재흡수↑, 신장 농축 | 증가 |
| **성장호르몬** | **뇌하수체 전엽** | 단백질 합성↑, 유리지방산 동원↑ | 증가 |
| **TSH** | **뇌하수체 전엽** | 갑상선 T3/T4 분비 조절 | - |
| **EPO(에리트로포이에틴)** | **신장** | 적혈구 생산 촉진 | 고지대 노출 시 증가 |

### [목록형] 운동 중 인슐린이 감소하는 이유

- 운동 중에는 혈당을 근육으로 흡수하게 하는 것보다 혈당을 유지하는 것이 중요
- 인슐린 감소 → 혈당 강하 억제 → 혈당 유지
- 나머지 호르몬(에피네프린, 코티솔, 성장호르몬, 글루카곤)은 **모두 증가**

> ⚠️ **주의** 에피네프린은 **부신수질**, 코티솔·알도스테론은 **부신피질** — '부신피질'에서 에피네프린이 나온다고 하면 오답. / 성장호르몬은 **뇌하수체 전엽** — 후엽이 오답으로 출제.

## 고유수용기 · 🔥🔥🔥

### [비교형] 근방추 vs 골지건기관

| 구분 | 근방추 (Muscle Spindle) | 골지건기관 (Golgi Tendon Organ, GTO) |
|------|----------------------|--------------------------------------|
| 위치 | 근육 내부 | **근육-건 접합부** |
| 감지 자극 | **근육 길이** (길이 변화, 속도) | **장력** (힘 생성량) |
| 신경 조절 | **감마운동뉴런** 지배 | 감마운동뉴런 지배 없음 |
| 반사 결과 | 주동근 **수축 촉진** (신전 반사) | 주동근 **수축 억제** (자가 억제) |
| 기능 목적 | 길이 변화 감지, 자세 유지 | 과도한 장력에서 **근육 손상 예방** |

> ⚠️ **주의** 근방추는 '길이', 골지건기관은 '장력(힘)'을 감지 — 반대로 외우면 전체 오답. / 골지건기관은 장력이 위험 수준에 이르면 수축을 **억제**하여 손상을 방지한다.

## 자율신경계 · 🔥🔥🔥

### [비교형] 교감신경 vs 부교감신경

| 구분 | 교감신경 | 부교감신경 |
|------|---------|-----------|
| 기능 | 흥분, 에너지 동원 | 억제, 회복, 소화 |
| 심박수 | **증가** | **감소** |
| 혈압 | 증가 | 감소 |
| 기관지 | 확장 | 수축 |
| 소화기계 | **억제** | 촉진 |
| 골격근 혈류 | **증가** | - |
| 내장기관 혈류 | **감소** | - |

- 운동 중 교감신경 활성화, 부교감신경 억제(철수)
- 부교감신경이 **활성화**되면 심박수가 **낮아짐** — 운동 중 부교감 활성으로 심박수 증가가 된다는 선지는 오답

> ⚠️ **주의** 운동 후 회복기에는 부교감신경이 재활성화되어 심박수가 낮아짐. 운동 중에는 교감신경이 지배적.

## 환경생리 (고지대 / 열 환경) · 🔥🔥🔥

### [비교형] 고지대 초기 반응 vs 장기 적응

| 구분 | 초기 반응 | 장기 적응 (수주~수개월) |
|------|----------|----------------------|
| 산소 분압 | 감소 | 감소 (변하지 않음) |
| 심박수 | **증가** (보상) | 정상 또는 감소 |
| 환기량 | **증가** (보상) | 증가 유지 |
| 헤모글로빈 포화도 | **감소** | 개선 |
| 적혈구 수 | 변화 없음 | **증가** (EPO 분비) |
| 모세혈관 밀도 | 변화 없음 | **증가** |
| 산소 운반 능력 | 감소 | **증가** |
| 유산소 능력 감소 폭 | 무산소보다 큼 | - |

- 고지대 장기 노출 시 심박출량은 **초기에 증가 → 장기적으로 낮아지거나 안정 수준**으로 돌아옴

### [비교형] 열 환경 반응

| 상황 | 반응 |
|------|------|
| 체온 **상승** | 피부혈관 확장, **발한(땀)** 증가, 심박수 보상적 증가 |
| 체온 **저하** | 피부혈관 수축, **떨림(Shivering)** 증가, 혈관 수축 |

### [비교형] 열 질환 단계

| 단계 | 심부체온 | 특징 |
|------|---------|------|
| 열경련 | 정상 이하 | 전해질 불균형, 근육 경련 |
| **열탈진** | **40°C 미만** | 극도의 피로, 탈수, 혼미 |
| **열사병** | **40°C 이상** | 체온조절 중추 마비, **생명 위험** |

### [목록형] 운동 중 열 손실 4기전

- **복사**: 전자기파로 열 방출
- **전도**: 접촉면으로 열 전달
- **대류**: 공기/물 흐름으로 열 방출
- **증발**: 땀이 기화되며 열 흡수 → **실내 운동 시 가장 주된 기전**

> ⚠️ **주의** 고지대 초기 반응에서 심박수는 '감소'가 아니라 **증가**. 심박수가 감소하면 오답. / 열탈진(40°C 미만)과 열사병(40°C 이상)을 반드시 구분.

## 건강 · 기술 체력 · 🔥🔥

### [목록형] 건강관련체력 5요소

1. **심폐지구력** — 오래달리기 측정
2. **근력** — 악력, 배근력 측정
3. **근지구력** — 팔굽혀펴기, 윗몸일으키기 측정
4. **유연성** — 앉아윗몸앞으로굽히기 측정
5. **신체구성** — 생체전기저항분석(BIA), 피부두겹법 측정

### [목록형] 기술관련체력 (건강체력 아님)

- 순발력 → 제자리높이뛰기, 제자리멀리뛰기
- 민첩성, 협응성, 평형성, 반응시간

> ⚠️ **주의** 제자리높이뛰기는 **순발력(기술체력)** 측정 — 건강체력이라고 하면 오답. / 건강관련체력 = 심폐지구력, 근력, 근지구력, 유연성, 신체구성 5가지가 정답.

## 환기 · 호흡계 · 🔥🔥🔥

### [비교형] 전도 영역 vs 호흡 영역

| 구분 | 전도 영역 (기관, 기관지 등) | 호흡 영역 (폐포) |
|------|--------------------------|----------------|
| 기능 | 공기 **정화**, **가온**, **가습** | **가스 교환** (O₂ 흡수 / CO₂ 배출) |
| 표면활성제(Surfactant) | 없음 | **생성** (폐포 표면장력 감소) |

### [단계-순서형] 환기량 조절 (운동 단계별)

| 단계 | 조절 요인 |
|------|---------|
| 운동 시작 **직전** | 대뇌의 **예비 반응** (심리적 기대감) → 환기량 증가 |
| 운동 **초기** | **신경적 요인** (고유수용기, 관절/근육 자극) |
| 운동 **중기** | **화학적 요인** (CO₂↑, H⁺↑, O₂↓) |
| 운동 **후 회복기** | 체내 수소이온·CO₂ 제거 위해 환기량 유지 |

- 경동맥체·대동맥체 = **말초 화학수용체** (중추가 아님)
- O₂ 분압 **감소** 또는 CO₂ 분압 **증가** 시 환기량 자극

> ⚠️ **주의** 경동맥체·대동맥체는 '중추 화학수용체'가 아니라 **말초 화학수용체**. / 운동 초기 환기 급증은 화학적 요인이 아닌 **신경적 요인**이 주된 원인.

## 섹션 3: 고급 노트 (변별력 문제)

---

### 에너지 대사 속도조절효소

| 에너지 경로 | 속도조절효소 |
|-----------|------------|
| ATP-PC 시스템 | **크레아틴 키나아제 (CK)** |
| 해당과정 | **포스포프럭토키나아제 (PFK)** |
| 크렙스 회로 | **이소시트르산 탈수소효소 (ICD)** |
| 전자전달계 | **사이토크롬 산화효소** |

❌ "ATP-PC 시스템의 속도조절효소는 포스포프럭토키나아제(PFK)" → PFK는 해당과정의 효소. ATP-PC는 **CK**
❌ "해당작용의 속도조절효소는 젖산 탈수소효소(LDH)" → LDH는 피루브산-젖산 전환 효소일 뿐, 속도조절효소가 아님

---

### METs · 운동량 계산

- **VO₂ 계산**: VO₂ (ml/min) = METs × 3.5 (ml·kg⁻¹·min⁻¹) × 체중(kg)
- **칼로리 계산**: 총 VO₂(L) × 5 Kcal/L

**예제** (80kg, 10METs, 10분):
- 분당 VO₂ = 10 × 3.5 × 80 = 2,800 ml/min = 2.8 L/min
- 총 VO₂ = 2.8 × 10 = 28 L
- 소비 칼로리 = 28 × 5 = **140 Kcal**

---

### 활동전위 단계 (과분극)

| 단계 | 이온 이동 | 전위 변화 |
|------|----------|---------|
| 안정막전위 | K⁺ 세포 내↑, Na⁺ 세포 외↑ | 약 -70mV |
| **탈분극** | **Na⁺ 유입** | -70mV → +30mV (급격히 상승) |
| **재분극** | **K⁺ 유출** | +30mV → -70mV 복귀 |
| **과분극** | K⁺ 채널 늦게 닫혀 **K⁺ 과유출** | 안정막보다 더 낮아짐 |

❌ "K⁺ 채널이 열리고 칼륨이 세포 외로 이동하며 음전하가 되는 단계 = 재분극" → 더 낮아지면 **과분극**

---

### 보어 효과 (Bohr Effect)

- 산소-헤모글로빈 해리 곡선의 **우측 이동** = 헤모글로빈의 산소 친화력 **감소** = 조직으로 산소 방출 **증가**

| 우측 이동 요인 | 
|-------------|
| 온도 **상승** |
| pH **감소** (산성) / H⁺ **증가** |
| CO₂ **증가** |
| 혈중 젖산 농도 **증가** |
| 2,3-DPG **증가** |

❌ "체온 상승은 헤모글로빈의 산소 친화력을 **높인다**" → 체온 상승 = 우측 이동 = 친화력 **감소**
❌ "고강도 운동 시 동-정맥 산소 차이는 안정 시보다 감소한다" → 근육이 더 많은 산소를 추출 → 동-정맥 산소차 **증가**

---

### EPOC (초과산소섭취량)

- EPOC = 운동 후 회복기에 산소 소비가 안정시보다 높게 유지되는 현상

| EPOC 유발 요인 | 내용 |
|--------------|------|
| 고갈된 PCr 재합성 | 인산크레아틴 저장량 회복 |
| 젖산 제거 (코리 사이클) | 간에서 젖산 → 포도당 재합성 |
| 체온 정상화 | 상승한 체온 하강 시 산소 소비 |
| 호르몬(에피네프린 등) 안정화 | 증가한 호르몬 농도 복귀 |

❌ "운동 중 증가한 크레아틴인산(PC)이 EPOC 요인이다" → 운동 중 PC는 **분해(감소)**됨. EPOC는 고갈된 PC를 **재합성**하기 위해 산소가 쓰이는 것

---

### 칼슘 항상성 호르몬

| 호르몬 | 분비 기관 | 혈중 Ca²⁺ 변화 | 작용 |
|--------|---------|--------------|------|
| **칼시토닌** | 갑상선 | 혈중 Ca²⁺ **높을 때** | 뼈에 Ca²⁺ 저장 → 혈중 Ca²⁺ ↓ |
| **부갑상선호르몬(PTH)** | 부갑상선 | 혈중 Ca²⁺ **낮을 때** | 뼈에서 Ca²⁺ 방출 → 혈중 Ca²⁺ ↑ |

❌ "칼시토닌 = 혈중 칼슘 낮을 때 분비" → 혈중 칼슘이 **높을 때** 분비하여 수준을 낮춤

---

### 카테콜아민 수용체 (α1, β1, β2)

| 수용체 | 위치/효과 |
|--------|---------|
| **α1** | 혈관 평활근 **수축** → 혈압↑ |
| **β1** | 심장 → 심박수↑, 수축력↑ |
| **β2** | 기관지 **확장** → 호흡 원활 |

- 카테콜아민(에피네프린·노르에피네프린) 분비처: **부신수질** (부신피질 아님)
- 교감신경 말단에서는 노르에피네프린 방출

❌ "카테콜아민은 부신피질에서 분비" → **부신수질**에서 분비
❌ "α1 수용체 결합 시 기관지 수축" → 기관지는 **β2** 수용체 결합 시 확장

---

### 심근산소소비량 (RPP, Double Product)

- **RPP = 심박수(HR) × 수축기 혈압(SBP)**
- 심장의 산소 소비 부담 지표
- 동일 VO₂에서 팔 운동 > 다리 운동 (팔이 더 작은 근육군 → 혈압 반응 급격)

❌ "심근산소소비량 = 심박수 × **1회 박출량**" → 1회 박출량이 아닌 **수축기 혈압**을 곱함

---

### 근절 구조: 수축 시 A밴드 불변

- **수축 시 변하는 것**: I 밴드(짧아짐), H 구역(짧아짐), 근절 전체 길이(짧아짐)
- **수축 시 변하지 않는 것**: **A 밴드** (마이오신 전체 길이 반영, 변화 없음), 액틴·마이오신 자체 물리적 길이

❌ "I 밴드의 길이는 수축 시 변하지 않는다" → I 밴드는 **짧아짐**
❌ "액틴과 마이오신의 길이가 짧아진다" → 단백질 자체 길이는 변하지 않고, 겹치는 정도만 깊어짐

### 고온 환경 운동능력 저하 4요인

| 요인 | 메커니즘 |
|------|---------|
| 글리코겐 고갈 가속 | 체온 상승 → 탄수화물 대사율 증가 |

❌ "심각한 탈수 없이도 1회 박출량은 변하지 않는다" → 고온에서 혈장량 감소로 **SV 감소**
