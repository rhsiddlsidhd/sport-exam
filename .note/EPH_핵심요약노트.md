# 운동생리학 핵심 요약 노트
### 생활스포츠지도사 2급 | 2019~2025년 기출 분석 기반 (140문항)

> **강사 한마디**: 운동생리학은 "근육→심혈관→에너지" 3대 축을 먼저 잡으면 된다. 전체 문항의 절반이 이 세 토픽에 집중되어 있고, 매년 반드시 나오는 패턴이 있다. 헷갈리는 선지는 대부분 "증가/감소"를 뒤집은 것이다.

## 섹션 2: 토픽별 핵심 개념

### 🏋️ 1. 근육 생리 (30회 — 최다 출제)

#### 근섬유 유형 비교 (매년 출제)

| 특성 | 지근 (Type Ⅰ) | 속근 (Type Ⅱ) |
|------|--------------|--------------|
| 수축 속도 | 느림 | 빠름 |
| 피로 저항성 | **높음** | **낮음** ← 시험 함정 |
| 미토콘드리아 | 많음 | 적음 |
| 모세혈관 밀도 | 높음 | 낮음 |
| ATPase 활성도 | 낮음 | 높음 |
| 주 에너지 | 산화적 인산화 | 해당과정 |
| 동원 순서 | 먼저 | 나중에 |

> **주의**: "속근은 지근보다 피로 저항성이 높다" → ❌ 반드시 틀린 선지

#### 근수축 과정 순서 (2019, 2025 출제)
```
ACh 방출(㉢) → 활동전위 발생(㉠) → Ca²⁺ 방출(SR, ㉡) → 필라멘트 활주(㉣)
```
- 트로포닌에 Ca²⁺ 결합 → 트로포마이오신 이동 → 마이오신 머리-액틴 결합 → 파워 스트로크

#### 근수축 유형

| 유형 | 설명 | 예시 |
|------|------|------|
| 단축성(구심성) | 근육 짧아지며 힘 생성 | 턱걸이 올라갈 때 |
| 신장성(원심성) | 근육 길어지며 힘 생성, **같은 속도에서 더 큰 힘** | 턱걸이 내려갈 때 |
| 등척성 | 길이 변화 없이 힘 생성 | 자세 유지 |
| 등속성 | 일정 속도 유지 | 등속성 운동기구 |

#### 고유수용기

| 기관 | 감지 | 반응 |
|------|------|------|
| 근방추 | 근육 **길이** 변화 | 신전반사 유발(흥분성) |
| 골지건기관(GTO) | 근육 **장력** | 수축 억제(억제성) |

#### 운동단위 (motor unit)
- 정의: 하나의 알파운동뉴런 + 그 신경이 지배하는 **모든 근섬유**
- 자극비율 낮음(신경 1개당 근섬유 수 적음) → **정교한 움직임** 가능 (눈, 손)
- Type Ⅰ 운동단위 → Type Ⅱ보다 **먼저** 동원(사이즈 원칙)

### ❤️ 2. 심혈관계 (28회)

#### 핵심 공식
```
심박출량(Q) = 1회박출량(SV) × 심박수(HR)
1회박출량(SV) = 확장기말 혈액량(EDV) - 수축기말 혈액량(ESV)
```

#### 1회박출량 증가 요인 vs 감소 요인

| 증가 요인 | 감소 요인 |
|-----------|-----------|
| 심근 수축력 증가 | **심박수 과도한 증가** ← 함정 |
| EDV(전부하) 증가 | 대동맥압(후부하) 증가 |
| 동맥혈압(후부하) 감소 | 탈수(혈장량 감소) |
| 근육펌프·호흡펌프에 의한 정맥환귀 증가 | — |

> **주의**: "심박수 증가 → 1회박출량 증가" → ❌ 과도한 심박수 증가는 충전 시간 부족으로 SV **감소**

#### 동방결절(SA node)
- 위치: **우심방** 벽
- 역할: 자발적 전기 자극 시작 = 페이스메이커(pacemaker)

#### 적혈구용적률(Hematocrit)
- 전체 혈액 중 **적혈구 비율**
- 지구성 트레이닝 → 혈장량 **증가** → 헤마토크릿 **상대적 감소** (희석 효과)
- 60% 이상 → 혈액 점성 증가 → 혈류 **저하**
- 성인 남성 > 성인 여성

#### 운동강도에 따른 혈류 재분배 (교감신경)
- 골격근 혈류 ↑
- 내장기관·소화기계 혈류 ↓ ← "소화기계 활동 증가" → ❌

#### 산소-헤모글로빈 해리 곡선

| 조건 | 곡선 이동 | 산소 친화력 |
|------|-----------|-------------|
| 체온↑ / pH↓(산성) / CO₂↑ | **우측 이동** | **감소** (산소 방출 쉬움) |
| 체온↓ / pH↑(알칼리) / CO₂↓ | **좌측 이동** | **증가** |

### ⚡ 3. 에너지 시스템 (21회)

#### 3대 에너지 시스템 비교

| 시스템 | 연료 | 지속시간 | ATP 생산 속도 | 산소 |
|--------|------|----------|---------------|------|
| ATP-PC | PCr(크레아틴인산) | ~10초 | **가장 빠름** | 불필요 |
| 무산소 해당과정 | 글리코겐 | ~2분 | 빠름 | 불필요 |
| 유산소 시스템 | 탄수화물·지방 | 장시간 | **가장 느림** ← 함정 | 필요 |

> **주의**: "유산소 시스템은 무산소보다 ATP 합성률이 빠르다" → ❌ **느리다**

#### 1개 포도당 유산소 분해 ATP 생성 (2023 TABLE 문제)

| 대사 과정 | 고에너지 생산 | ATP 누계 |
|-----------|-------------|---------|
| 해당작용 | 2 ATP | 2 |
| | 2 NADH | 7 |
| 피루브산 → 아세틸CoA | 2 NADH | 12 |
| **크랩스회로(TCA)** | 2 ATP | 14 |
| | 6 NADH | 29 |
| | 2 FADH₂ | **32** |
| **합계** | — | **32 ATP** |

> NADH = 2.5 ATP, FADH₂ = 1.5 ATP 적용

#### 지방 대사 (베타산화)
- 중성지방 → **유리지방산** + 글리세롤 (리파아제에 의해 분해)
- 유리지방산 → 미토콘드리아 내 **베타(β)산화** → 아세틸CoA → TCA 회로

#### 젖산역치(LT)와 코리사이클
- 젖산역치 발생 원인: 산소 부족, 속근 동원 증가, 무산소 해당 의존 증가
- **코리사이클**: 젖산 → 간에서 포도당 재합성 → 혈중 젖산 **낮추는** 기제
  > "코리사이클 증가 → 젖산역치 원인" → ❌

#### 운동강도에 따른 에너지원 전환
- 저강도: 지방 비율 높음 (RER ≈ 0.7)
- 고강도: 탄수화물(근글리코겐) 비율 높음 (RER → 1.0 이상)
- 훈련 선수: 같은 강도에서 비훈련자보다 지방 사용 비율 높음

### 🫁 4. 호흡·가스교환 (16회)

#### 폐환기 계산 공식
```
분당환기량(Ve) = 1회호흡량(TV) × 호흡수(RR)
폐포환기량(Va) = (TV - 사강용적) × RR
```
> 사강(Dead Space) = 약 150mL (가스교환 불가 영역)

#### 호흡교환율(RER)

| RER 값 | 주 에너지원 |
|--------|------------|
| 0.7 | 지방 |
| 0.85 | 혼합 (지방+탄수화물) |
| 1.0 | 탄수화물 |
| >1.0 | 고강도 운동 (과잉 CO₂ 생성) |

#### 운동 중 환기량 변화
- 운동 시작 **직전**: 기대감만으로 환기량 증가 (신경성 요인)
- 운동 **초기**: 신경성 자극 주도 (화학수용기 반응 ← 주 요인 ❌)
- 운동 강도↑: 1회호흡량↑ **후** 호흡수↑ (1회호흡량이 먼저 증가)
- 회복기: 체내 H⁺·CO₂ 농도와 관련

#### 가스교환 방향
- 폐포 → 폐모세혈관: **산소** 확산 (폐동맥혈의 낮은 PO₂ → 확산 촉진)
- 근육 → 모세혈관: **이산화탄소** 확산

#### 주요 호르몬 분비처와 기능

| 호르몬 | 분비처 | 운동 중 기능 |
|--------|--------|-------------|
| **에피네프린** | 부신 **수질** | 글리코겐·지방 분해↑, 심박수·수축력↑ |
| 노르에피네프린 | 부신 수질 | 혈관 수축, 혈압↑ |
| **인슐린** | 췌장 β세포 | 혈당↓ (포도당 세포 흡수↑) — 운동 중 **감소** |
| **글루카곤** | 췌장 α세포 | 혈당↑ (글리코겐 분해↑) |
| 코르티솔 | 부신 **피질** | 단백질·지방 분해↑, 항염증 |
| 성장호르몬(GH) | 뇌하수체 전엽 | 지방 분해↑, 단백질 합성↑ |
| 갑상선자극호르몬(TSH) | 뇌하수체 전엽 | T3·T4 분비 조절 |
| 알도스테론 | 부신 피질 | Na⁺ 재흡수↑ → 체액 보존 |
| ADH(항이뇨호르몬) | 뇌하수체 후엽 | 수분 재흡수↑ |
| EPO(에리트로포이에틴) | 신장 | 적혈구 생성 촉진 (고지대 적응) |

#### 1시간 이내 중강도 운동 시 혈중 농도 변화
- **점차 감소**: 인슐린 ← 정답 포인트
- 점차 증가: 에피네프린, 코르티솔, 성장호르몬

### 🧠 6. 신경계 (14회)

#### 안정막전위와 이온 분포

| 위치 | Na⁺ | K⁺ |
|------|-----|-----|
| 세포 **외액** | **높음** (150mM) | 낮음 (5mM) |
| 세포 **내액** | 낮음 | **높음** |

> Na⁺/K⁺ 펌프: Na⁺ **3개** 밖으로, K⁺ **2개** 안으로

#### 자율신경계 비교

| | 교감신경 | 부교감신경 |
|-|---------|-----------|
| 성격 | 흥분성 | 억제성 |
| 운동 시 | 활성화 | 억제 |
| 심박수 | ↑ | ↓ |
| 소화기계 | ↓ | ↑ |
| 골격근 혈류 | ↑ | — |

#### 소뇌 기능 (2023 출제)
- 빠른 동작의 정확한 수행 통합 조절
- 고유수용기 정보 활용해 동작 수정
- (주의: 골격근 운동 조절의 **최종 단계는 대뇌피질**)

#### 신경-근 접합부(NMJ)
- 아세틸콜린(ACh) → 근세포막 수용체 결합 → 탈분극 → 활동전위 → Ca²⁺ 방출

### 🌡️ 7. 체온·체액조절 (6회)

- 탈수 → 혈장량↓ → 정맥환귀↓ → SV↓ → **심박수 보상적 증가** → 심혈관계 표류(drift)
- 수중 운동 시 체온유지 요인: 체지방량, 운동 강도, 물의 온도 (폐활량 ❌)
- 고온다습 운동: ADH↑(체액 보존), 알도스테론↑(Na⁺ 재흡수)

### 🏃 8. 트레이닝 적응 (6회)

#### 장기간 유산소 트레이닝 적응

| 변화 | 방향 |
|------|------|
| 안정시 심박수 | ↓ |
| 최대 1회박출량 | ↑ |
| 최대 심박출량 | ↑ |
| 최대산소섭취량(VO₂max) | ↑ |
| 안정시 1회박출량 | **↑** ← "감소"가 함정 선지 |
| 혈장량 | ↑ (헤마토크릿 상대적 감소) |
| 미토콘드리아 밀도 | ↑ |
| 모세혈관 밀도 | ↑ |
| 최대 동정맥산소차 | ↑ |

#### 저항성 트레이닝 적응
- 근비대: 근형질·근원섬유 증가, 속근 단면적 증가
- 미토콘드리아 밀도 증가 → **저강도 저항성 훈련** ❌ (유산소 특징)
- 저강도 고반복(20~25회) → **근지구력** 향상

#### VO₂max 공식
```
VO₂max = 최대1회박출량 × 최대심박수 × 최대동정맥산소차
```
- 지구성 훈련으로 증가: 최대1회박출량(㉠), 최대동정맥산소차(㉢)
- 최대심박수는 훈련으로 변하지 않음

## 섹션 3: 함정 선지 총정리

| ❌ 틀린 선지 | ✅ 올바른 내용 |
|------------|--------------|
| 유산소 시스템은 무산소보다 ATP 합성률이 빠르다 | **느리다** (지속시간이 길 뿐) |
| 속근은 지근보다 피로 저항성이 높다 | 지근이 더 높음 |
| 심박수 증가 → 1회박출량 증가 | 과도한 심박수↑ → SV **감소** (충전시간 부족) |
| 코리사이클 증가 → 젖산역치 원인 | 코리사이클은 젖산을 **낮추는** 기제 |
| 지근이 속근으로 전환된다 | 전환 불가. 속근이 지근 특성을 일부 가질 수는 있음 |
| 수중 운동 체온유지 요인 — 폐활량 | 폐활량은 관련 없음 |
| 안정시 1회박출량 감소 (유산소 적응) | 안정시 SV **증가** |
| 미토콘드리아 밀도 증가 (저항성 트레이닝) | 저항성보다 **유산소 트레이닝**에서 주로 발생 |
| 지구성 훈련 → 헤마토크릿 증가 | 혈장량↑로 오히려 **상대적 감소** |
| 소뇌 = 골격근 운동 조절의 최종 단계 | 최종 단계는 **대뇌피질** |
| 체온↑ → 산소-헤모글로빈 곡선 좌측 이동 | **우측** 이동 (산소 친화력 감소) |
| 소화기계 활동이 증가한다 (교감신경) | 교감신경 → 소화기계 **억제** |
| 운동 초기 환기량의 주 요인 = 화학수용기 | 초기는 **신경성 자극**이 주 요인 |
| 인슐린 혈중 농도 — 중강도 운동 중 증가 | 운동 중 **감소** |
| 특이장력 = 근력 × 근횡단면적 | 특이장력 = 근력 **/** 근횡단면적 |

## 섹션 5: 압축 암기 공식

```
■ 심박출량 = SV × HR
■ SV = EDV - ESV
■ VO₂max = 최대SV × 최대HR × 최대동정맥산소차
■ 분당환기량(Ve) = TV × RR
■ 폐포환기량(Va) = (TV - 사강) × RR
■ 특이장력 = 근력 / 근횡단면적
■ 근파워 = 힘 × 수축속도 (P = F × v)
■ RPP(심근산소소비량) = HR × 수축기혈압(SBP)
■ 포도당 유산소 ATP = 32개 (NADH×2.5 + FADH₂×1.5 + ATP)
■ Na⁺/K⁺ 펌프 = Na⁺ 3개 OUT / K⁺ 2개 IN

■ 에너지 속도: ATP-PC > 무산소해당 > 유산소
■ 근섬유 동원 순서: TypeⅠ(지근) → TypeⅡa → TypeⅡx
■ 근수축 순서: ACh → 활동전위 → Ca²⁺ → 트로포닌 → 트로포마이오신 이동 → 활주
■ RER: 지방=0.7 / 혼합=0.85 / 탄수화물=1.0
■ 교감: 흥분·운동 / 부교감: 안정·억제 (심박수·소화 반대)
```
