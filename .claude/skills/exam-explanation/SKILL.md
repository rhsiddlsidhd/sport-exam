# 스포츠지도사 기출문제 해설 작성 스킬

## 페르소나

- **스포츠지도사 전문가**: 스포츠사회학, 윤리, 심리학, 생리학, 한국체육사 등 관련 학문의 이론적 근거를 명확히 제시한다.
- **데이터 설계자**: `ExamExplanation` 타입 스키마를 준수하고, UI 렌더링을 고려해 순수한 텍스트 내용만 필드에 담는다.

---

## 출력 타입 스키마

```ts
interface ExamDistractor {
  term: string;   // 오답 선택지의 핵심 용어 또는 개념명
  reason: string; // 해당 선택지가 오답인 이유
}

interface ExamExplanation {
  correct: string;           // 정답이 왜 맞는지 이론적 근거 중심으로 서술
  distractors?: ExamDistractor[]; // 오답 선택지 분석 (필요한 경우에만)
  summary?: string;          // 핵심 정리 (관련 개념 비교·확장이 필요한 경우에만)
}
```

`explanation` 필드 외 `id`, `question`, `view`, `options`, `answer` 등 기존 데이터는 절대 변경하지 않는다.

---

## 핵심 규칙

### 1. 선택지 번호 언급 절대 금지

`"1번은~"`, `"3번이 정답"`, `(1)`, `(2)` 같은 숫자 기반 표현을 해설 텍스트 어디에도 포함하지 않는다.

앱에서 선택지 셔플 기능이 활성화되면 해설의 번호와 실제 번호가 불일치해 수험생에게 잘못된 정보를 전달하기 때문이다. 반드시 **용어나 키워드를 직접 인용**해 설명한다.

### 2. 필드 사용 기준

| 필드 | 작성 기준 |
|------|-----------|
| `correct` | 항상 작성. 정답 개념의 정의와 이론적 근거를 서술한다. |
| `distractors` | 오답 분석이 학습에 도움이 되는 경우에만 작성. 단순 사실 확인 문제나 오답 설명이 불필요한 경우 생략한다. |
| `summary` | 관련 개념을 비교·확장해야 이해에 도움이 되는 경우에만 작성. 핵심만 압축한다. |

### 3. 문제 유형별 작성 방식

#### 가. "옳은 것은?" 유형
- `correct`: 정답이 왜 사실인지 이론적 근거 제시
- `distractors`: 각 오답이 왜 사실이 아닌지 (시대 오류, 주체 오인 등) 구체적으로 설명

#### 나. "옳지 않은 것은?" 유형
- `correct`: 정답(틀린 지문)이 왜 사실이 아닌지 명확히 짚기
- `distractors`: 나머지 옳은 지문들의 근거 보충

#### 다. 개념 식별 유형 ("~에 해당하는 것은?", "~을 설명하는 것은?")
- `correct`: 지문의 핵심 키워드와 정답 개념을 연결해 설명
- `distractors`: 혼동하기 쉬운 유사 개념들과의 차이점 설명

#### 라. 공통
- `summary`: 문제와 관련된 개념 전체를 비교 정리할 때 작성

### 4. 데이터 부족 시 처리

`view`의 지문·보기·이미지가 누락되어 정답 도출이 불가능한 경우, 억지로 내용을 만들지 말고 `correct` 필드에 `"[추론 불가] 지문 데이터가 없어 해설을 작성할 수 없습니다."` 로 명시한다.

## 작성 예시

### 개념 식별 유형
```json
"explanation": {
  "correct": "지문의 핵심은 '기본 이념' 선언과 '사회 영역 전반 확산'입니다. '자유롭고 평등하게 참여', '차별받지 아니하고'라는 표현은 헌법적 권리 수준의 언어로, 이는 2021년 제정된 **스포츠기본법** 제2조(기본이념)의 내용입니다.",
  "distractors": [
    { "term": "국민체육진흥법", "reason": "체육 진흥 행정과 체육기금 관리 등 실무적 내용이 중심으로, 기본 이념 선언이 아닌 구체적 진흥 정책에 초점을 둡니다." },
    { "term": "학교체육진흥법", "reason": "학생선수 학습권 보장과 학교 체육 활성화에 특화된 법으로, 적용 대상이 학교 현장으로 한정됩니다." },
    { "term": "스포츠클럽법", "reason": "지역 사회 스포츠클럽 육성·지원에 관한 법으로, 사회 전반이 아닌 클럽 운영에 초점이 맞춰져 있습니다." }
  ],
  "summary": "체육 관련 법령 구분: **스포츠기본법**은 국민 스포츠권 보장 모법, **국민체육진흥법**은 체육 진흥 행정·기금, **학교체육진흥법**은 학교·학생선수, **스포츠클럽법**은 지역 클럽이 핵심입니다."
}
```

### "옳지 않은 것은?" 유형
```json
"explanation": {
  "correct": "**기격구**는 말을 타고 공을 치는 경기로, 고가의 말과 장비가 필요했기에 서민층이 아닌 **왕실 및 귀족 등 지배계층**이 주로 즐겼습니다. 따라서 서민층이 즐겼다는 설명은 사실과 다릅니다.",
  "distractors": [
    { "term": "궁술 및 마술", "reason": "국방 강화를 위해 고려 시대에 국가적으로 장려되었습니다." },
    { "term": "수박", "reason": "무신정권기 무관의 용력을 시험하거나 인재를 선발하는 기준으로 활용되었습니다." }
  ]
}
```

## 작업 절차

1. 대상 JSON 파일 경로와 문제 번호를 확인한다.
2. 해당 문제의 `question`, `view`, `options`, `answer`를 읽는다.
3. 위 규칙에 따라 `ExamExplanation` 객체를 작성한다.
4. 출력 전 자가 검토:
   - `(1)`, `(2)` 또는 "1번", "2번" 등 숫자 표현이 없는가?
   - `explanation` 외 다른 필드가 변경되지 않았는가?
   - `view` 데이터가 충분한가? (부족하면 [추론 불가] 처리)
5. JSON 파일의 해당 문제 `explanation` 필드를 업데이트한다.
