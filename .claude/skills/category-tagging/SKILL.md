# category-tagging 스킬

기출 JSON 파일에 `category` 필드를 추가한다. 토큰 절약을 위해 question + view만 추출하여 분류한 뒤, 패치 스크립트로 삽입한다.

---

## ⛔ 절대 규칙

| 번호 | 규칙 |
|------|------|
| 1 | `src/data/exam/` JSON 파일을 직접 Read/Write하지 않는다. 반드시 스크립트를 통해서만 접근한다. |
| 2 | 추론이 불확실한 경우 억지로 분류하지 않고 **⚠️ 추론 불가**로 보고한다. |
| 3 | 패치 전 반드시 사용자에게 분류 결과를 보고하고 확인을 받는다. |
| 4 | 카테고리는 해당 과목의 확정된 체계에서만 선택한다. 임의 생성 금지. |

---

## 워크플로우

### 1단계: 추출

```bash
node scripts/extract-questions.mjs --subject {과목코드} --year {연도}
```

- question + view만 추출 (explanation, options, answer 제외 → 토큰 절약)
- 이미 category가 있는 문제는 결과에 포함되어 표시됨

### 2단계: 분류 보고

추출 결과를 보고 아래 형식으로 보고한다:

| # | 근거 키워드 | 분류 결과 |
|---|------------|----------|
| Q1 | ... | ... |
| Q2 | ⚠️ 추론 불가 | — |

- view 없이 question 텍스트만으로 특정 불가한 경우 → **⚠️ 추론 불가**로 표시
- 사용자가 추론 불가 항목에 대해 카테고리를 직접 지정하면 반영

### 3단계: 사용자 확인

사용자가 수정/확인하면 확정 매핑을 완성한다.

### 4단계: 패치

```bash
node scripts/patch-categories.mjs \
  --subject {과목코드} \
  --year {연도} \
  --categories '{"1":"카테고리명","2":"카테고리명",...}'
```

- questionNumber(문자열)를 키로 사용
- 변경된 문제 수와 저장 완료 여부를 출력으로 확인

## 과목별 카테고리 체계

### SSO (스포츠사회학)

- 스포츠사회학 개론
- 스포츠와 정치
- 스포츠와 경제
- 스포츠와 미디어
- 스포츠와 사회계층
- 스포츠 세계화
- 스포츠 일탈
- 스포츠 사회화
- 스포츠와 교육

### SET (스포츠윤리)

- 스포츠윤리 개론
- 스포츠와 윤리
- 경쟁과 페어플레이
- 도핑과 금지약물
- 스포츠와 폭력
- 스포츠와 젠더
- 스포츠와 인종
- 스포츠와 장애
- 스포츠 환경윤리
- 스포츠 조직과 윤리

### SPS (스포츠심리학)

- 스포츠심리학 개론
- 운동동기
- 각성·불안·스트레스
- 심리기술훈련
- 주의집중
- 자신감
- 집단역학
- 운동학습
- 운동과 심리건강
- 스포츠상담
- 스포츠 공격성

### KHS (한국체육사)

- 체육사 개론
- 부족국가시대 체육
- 삼국시대 체육
- 고려시대 체육
- 조선시대 체육
- 개화기 체육
- 일제강점기 체육
- 광복 이후 체육

### EPH (운동생리학)

- 에너지 대사
- 골격근
- 심폐계
- 혈액과 순환
- 트레이닝 적응
- 신경계
- 내분비계
- 체력
- 환경과 운동

## 스크립트 참조

| 스크립트 | 역할 |
|---------|------|
| `scripts/extract-questions.mjs` | question + view 추출 (--subject, --year 필터) |
| `scripts/patch-categories.mjs` | category 필드 패치 (--subject, --year, --categories) |

---

## 작업 완료 기준

- 대상 파일의 모든 문제에 category 필드가 삽입됨
- 패치 스크립트 출력에서 "스킵: 0문제" 또는 의도한 스킵 수 확인
- 추론 불가 항목이 있을 경우 사용자가 직접 지정하여 전부 해소됨
