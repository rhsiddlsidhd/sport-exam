# 00. Project Overview

## 프로젝트 목적

생활체육지도자 2급 필기시험 기출문제를 연도별로 풀어볼 수 있는 모바일 우선 웹 애플리케이션이다.

## 대상 사용자

생활체육지도자 2급 자격증 취득을 준비하는 수험생.

## 핵심 기능

- 과목 선택 (스포츠사회학, 스포츠교육학, 스포츠심리학, 한국체육사, 운동생리학)
- 연도별 기출문제 선택
- 한 문제씩 풀기 (선택지 셔플, 진행 현황 표시)
- 전체 답안 제출 후 채점 결과 확인

## 기술 스택

| 구분       | 기술                         |
| ---------- | ---------------------------- |
| 프레임워크 | React 19 + TypeScript + Vite |
| 라우팅     | React Router v7              |
| 스타일링   | Tailwind CSS v4 + shadcn/ui  |
| 유틸리티   | clsx                         |
| 데이터     | 정적 JSON 파일 (동적 import) |

## 로컬 실행

```bash
npm install
npm run dev
```

## 과목 코드 매핑

| SubjectCode | 과목명       |
| ----------- | ------------ |
| `SOC`       | 스포츠사회학 |
| `ETH`       | 스포츠교육학 |
| `PSY`       | 스포츠심리학 |
| `HIS`       | 한국체육사   |
| `PHY`       | 운동생리학   |
