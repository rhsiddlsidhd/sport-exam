# 00. Project Overview

## 동기/배경

생활체육지도자 2급 전용 기출문제 풀이 앱이 없어, 직접 모바일에서 편하게 학습할 수 있는 환경을 만들기 위해 개발했다.

## 프로젝트 목적

생활체육지도자 2급 필기시험 기출문제를 과목별·연도별로 풀어볼 수 있는 모바일 우선 웹 애플리케이션이다.

## 대상 사용자

생활체육지도자 2급 자격증 취득을 준비하는 수험생.

## 핵심 기능

- 과목 선택 (스포츠사회학, 스포츠윤리, 스포츠심리학, 한국체육사, 운동생리학)
- 연도별 기출문제 선택 (과목 선택 후 연도 선택)
- 한 문제씩 풀기 (선택지 셔플, 진행 현황 표시)
- 전체 답안 제출 후 채점 결과 확인
  - 채점 결과 화면에서 틀린 문제만 모아보기
- 오답 노트
  - 틀린 문제를 저장해서 나중에 보기
  - 날짜별 자동 그룹화 및 필터링 제공 (복습 시점 관리)

## 배포/운영 계획

- **배포 플랫폼**: Github 레포지토리 연동 기반 Vercel 배포
- **운영 형태**:누구나 자유롭게 접속하여 학습할 수 있는 오픈소스 기반 공개 웹 서비스
- **데이터 관리**: 별도의 서버 없이 브라우저 저장소를 활용한 클라이언트 사이드 데이터 관리

## 디자인/UX 톤

- 카드 중심의 심플한 모바일 UI
- 학습에 집중할 수 있는 미니멀한 인터페이스를 지향
- 학습의 피로도를 낮추는 깔끔하고 밝은 톤 유지
