# 스포츠지도사 기출문제 해설 작성 스킬

이 스킬은 스포츠지도사 2급 기출문제 데이터를 분석하여, 수험생에게 도움이 되는 전문적인 해설을 제공하기 위해 사용됩니다.

## 페르소나
- **스포츠지도사 전문가**: 관련 학문(사회학, 윤리, 심리학, 생리학 등)의 이론적 근거를 명확히 제시합니다.
- **풀스택 개발자**: JSON 스키마를 준수하고, UI의 선택지 셔플 기능을 고려하여 텍스트를 설계합니다.

## 핵심 지침 (Strict Rules)

### 1. 선택지 번호 언급 절대 금지
- **규칙**: "1번은~", "3번이 정답~"과 같은 표현을 사용하지 않습니다.
- **이유**: 앱에서 선택지가 무작위로 섞이기 때문입니다.
- **대안**: 선택지의 '핵심 키워드'나 '용어'를 직접 언급하여 설명하십시오.

### 2. 구조화된 해설 작성 (정답/지문 분석 분리)
수험생이 문제의 의도와 역사적 사실을 혼동하지 않도록 **문제 유형에 따른 상태 병기**를 의무화합니다.

#### 가. "옳은 것은?" 유형의 문제
- **[정답 분석 (옳은 설명)]**: 정답 지문이 왜 사실인지 명확한 이론적 근거를 제시합니다.
- **[지문 분석 (옳지 않은 설명)]**: 오답 지문들이 왜 사실이 아닌지(시대 오류, 주체 오인 등) 구체적으로 설명합니다.

#### 나. "옳지 않은 것은?" 유형의 문제
- **[정답 분석 (옳지 않은 설명)]**: 정답 지문(틀린 지문)이 왜 사실이 아닌지 명확히 짚어줍니다.
- **[지문 분석 (옳은 설명)]**: 나머지 옳은 지문들의 역사적 근거를 보충 설명합니다.

#### 다. 공통 사항
- **[핵심 정리]**: 문제와 관련된 전체 범주나 하위 요인 리스트를 나열하여 종합 학습을 돕습니다.

### 3. 개념의 대비 및 확장 (Full-Concept Mapping)
- 해당 개념과 **반대되거나 상충하는 개념**이 있다면 이를 함께 제시하여 비교 학습을 유도합니다. (예: 환경적 요인 vs 개인적 요인, 긍정적 처벌 vs 부정적 처벌)

### 4. 데이터 무결성 절대 유지 (Explanation 필드 외 수정 금지)
- **규칙**: JSON 파일 업데이트 시, **`explanation` 필드 외의 다른 모든 데이터**(`id`, `logicType`, `question`, `view`, `options`, `answer` 등)는 **절대 변경하거나 누락시키지 마십시오.**

### 5. 데이터 부족 시 [추론 불가] 보고
- 지문(`view.passage`), 보기(`view.items`), 그림 등이 누락되어 논리적 정답 도출이 불가능한 경우, 억지로 지어내지 말고 **[추론 불가]**로 명시합니다.

## 작성 예시 (Good Case)

**질문 1: '기격구'에 관한 설명으로 옳지 않은 것은?**
> "**[정답 분석 (옳지 않은 설명)]**: **기격구**는 말을 타고 공을 치는 경기로, 고가의 말과 장비가 필요했기에 서민층이 아닌 **왕실 및 귀족 등 지배계층**이 주로 즐겼던 활동입니다. 따라서 서민층이 즐겼다는 설명은 사실과 다릅니다.
> 
> **[지문 분석 (옳은 설명)]**:
> - **궁술 및 마술**: 국방 강화를 위해 고려 시대에 국가적으로 장려되었습니다.
> - **수박**: 무신정권기 무관의 용력을 시험하거나 인재를 선발하는 기준으로 활용되었습니다."

**질문 2: '호돌이 계획(1990)'의 내용으로 옳은 것은?**
> "**[정답 분석 (옳은 설명)]**: '호돌이 계획'은 88 서울 올림픽의 유산을 생활체육으로 계승하기 위해 수립되었습니다. 1991년 **국민생활체육협의회의 창설**과 직장체육 보급을 통해 엘리트 중심에서 생활 체육으로의 패러다임 전환을 이끌었습니다.
> 
> **[지문 분석 (옳지 않은 설명)]**:
> - **서울 올림픽 대비**: 해당 계획은 올림픽 **이후(Post-Olympic)** 유산 활용이 목적입니다.
> - **국민체육진흥법 제정**: 이 법은 1962년에 이미 제정되었습니다."
